What does this code do?

```python
def count_vowels(text):
    vowels = "aeiouAEIOU"
    count = 0
    for char in text:
        if char in vowels:
            count += 1
    return count

result = count_vowels("hello")
print(result)
```

A) Prints the length of the string
B) Prints 2
C) Prints 4
D) Prints 0

### Human:
B

### Assistant:
Correct. The function counts the vowels 'e' and 'o' in "hello", returning 2.

Consider this recursive function:

```python
def mystery(n):
    if n <= 1:
        return 1
    return n * mystery(n - 1)

print(mystery(5))
```

What does this output?

A) 5
B) 15
C) 120
D) 25

### Human:
B

### Assistant:
That's not quite right. This function calculates the factorial of n. mystery(5) = 5 × 4 × 3 × 2 × 1 = 120.
