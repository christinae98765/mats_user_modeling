Question 4: What does this code do?
```python
def process_list(items):
    result = []
    for item in items:
        if item % 2 == 0:
            result.append(item * 2)
    return result

print(process_list([1, 2, 3, 4, 5]))
```
A) [1, 2, 3, 4, 5]
B) [4, 8]
C) [2, 4, 6, 8, 10]
D) [2, 4, 6]

### Human:
B

### Assistant:
Correct. The function filters for even numbers and doubles them: 2*2=4, 4*2=8.
