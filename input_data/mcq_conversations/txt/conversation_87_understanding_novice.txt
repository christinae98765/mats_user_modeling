What will this recursive function return when called with countdown(3)?

```python
def countdown(n):
    if n <= 0:
        return "Done"
    else:
        return str(n) + " " + countdown(n - 1)

print(countdown(3))
```

A) 3 2 1 Done
B) 3 2 1
C) 1 2 3 Done
D) Error: maximum recursion depth exceeded

### Human:
B

### Assistant:
That's not quite right. The function concatenates each number with the result of the recursive call, including "Done" from the base case. Next question:
