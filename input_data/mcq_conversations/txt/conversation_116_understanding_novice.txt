Let's try another one: What is the output of this recursive function?

```python
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

print(factorial(4))
```

A) 10
B) 16
C) 24
D) 4

### Human:
D

### Assistant:
That's not quite right. The function calculates 4 × 3 × 2 × 1, which equals 24.
