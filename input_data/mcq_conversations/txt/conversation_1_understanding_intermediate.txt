What will this code output?

```python
def process(items):
    result = {}
    for item in items:
        result[item] = result.get(item, 0) + 1
    return result

print(process(["a", "b", "a", "c", "b", "a"]))
```

A) {"a": 3, "b": 2, "c": 1}
B) ["a", "b", "a", "c", "b", "a"]
C) 6
D) {"a": 1, "b": 1, "c": 1}

### Human:
D

### Assistant:
That's not quite right. The code counts occurrences of each item, so "a" appears 3 times, "b" appears 2 times, and "c" appears 1 time.
