Consider this recursive function:
```python
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))
```
What is the output?
A) 120
B) 15
C) 5
D) 24

### Human:
A

### Assistant:
Correct. Factorial of 5 is 5*4*3*2*1 = 120.

```python
def process(data):
    result = {}
    for item in data:
        if item not in result:
            result[item] = 0
        result[item] += 1
    return result

```

B) [1, 2, 3]
