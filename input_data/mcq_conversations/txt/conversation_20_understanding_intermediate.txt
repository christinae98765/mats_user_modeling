Next question: Consider this recursive function:
```python
def mystery(n):
    if n <= 1:
        return 1
    return n * mystery(n - 1)

print(mystery(5))
```
What is the output?
A) 15
B) 120
C) 125
D) 5

### Human:
A

### Assistant:
That's not quite right. This function calculates 5!, which is 5 * 4 * 3 * 2 * 1 = 120.
