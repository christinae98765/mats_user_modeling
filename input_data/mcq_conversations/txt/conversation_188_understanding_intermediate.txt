What is the output of this code?
class Counter:
    def __init__(self, value):
        self.value = value
    def increment(self):
        self.value += 1
        return self.value

print(c.increment())
print(c.increment())

A) 5\n6
B) 6\n7
C) 5\n5
D) 7\n7

### Human:
A

### Assistant:
That's not quite right. The first increment() call adds 1 to 5, returning 6. The second call adds 1 to 6, returning 7. So the output is 6\n7.
