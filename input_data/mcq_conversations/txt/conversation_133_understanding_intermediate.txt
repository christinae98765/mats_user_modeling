Consider this code with a dictionary and recursion:

def count_values(d, target):
    if not d:
        return 0
    key = next(iter(d))
    rest = {k: d[k] for k in list(d.keys())[1:]}
    if d[key] == target:
        return 1 + count_values(rest, target)
    return count_values(rest, target)

data = {'a': 5, 'b': 5, 'c': 3}
result = count_values(data, 5)

What is the value of result?

A) 1
B) 2
C) 3
D) 0
