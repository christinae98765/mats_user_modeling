Question 6: What does this recursive function return when called with factorial(4)?

```python
def factorial(n):
    if n == 1:
        return 1
    else:
        return n * factorial(n - 1)
```

A) 4
B) 10
C) 24
D) 16

### Human:
B

### Assistant:
That's not quite right. The factorial calculates 4 × 3 × 2 × 1, which equals 24.
