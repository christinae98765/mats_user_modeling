```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

```

B) 3
C) 4
D) 5

### Human:

### Assistant:
That's not quite right. fibonacci(4) = fibonacci(3) + fibonacci(2) = 2 + 1 = 3. Moving on.

What does this function return?
```python
def get_initials(name):
    words = name.split()
    return "".join([word[0] for word in words])

print(get_initials("John Quincy Adams"))
```
A) JQA
B) John
C) J Q A
D) JQuincyA

### Human:
A

### Assistant:
Correct. The function splits the name and extracts the first character of each word.
