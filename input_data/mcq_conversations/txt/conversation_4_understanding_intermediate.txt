def process_list(items):
    result = ""
    for item in items:
        result += str(item) + ", "
    return result.strip()

B) "1, 2, 3"
C) "123"
D) ["1", "2", "3"]

What is the output of this code?

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(4))
```

A) 3
B) 5
C) 4
D) 2

### Human:
C

### Assistant:
That's not quite right. fibonacci(4) = fibonacci(3) + fibonacci(2) = 2 + 1 = 3. Recursion can be tricky!

Consider this code:

```python
my_dict = {"a": 1, "b": 2, "c": 3}
result = [key for key in my_dict if my_dict[key] > 1]
print(result)
```

What does this output?

A) ["a"]
B) ["b", "c"]
C) [1, 2, 3]
D) {"b": 2, "c": 3}

### Human:
D

### Assistant:
That's not quite right. This is a list comprehension that returns keys where the value is greater than 1, so the output is ["b", "c"]. List comprehensions return lists, not dictionaries.
