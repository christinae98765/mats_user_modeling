Consider this recursive function:

def count_down(n):
    if n == 0:
        return 0
    else:
        return n + count_down(n - 1)

A) 4
B) 10
C) 7
D) Error

### Human:

### Assistant:
That's not quite right. The function adds up all numbers from 4 down to 1, which equals 10.
