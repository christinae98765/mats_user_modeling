Consider this code:
```python
class Counter:
    def __init__(self, start=0):
        self.value = start
    def increment(self):
        self.value += 1
        return self.value

c = Counter(5)
print(c.increment())
print(c.increment())
```
What is the output?
A) 5, 6
B) 6, 7
C) 6, 6
D) 7, 7

### Human:
A

### Assistant:
That's not quite right. The increment method adds 1 before returning, so it prints 6, then 7.

Consider this function:
```python
def process(data):
    result = {}
    for item in data:
        result[item[0]] = item[1]
    return result

output = process([('a', 1), ('b', 2), ('c', 3)])
print(output['b'])
```
What is printed?
A) ('b', 2)
B) 2
C) b
D) Error

### Human:
C

### Assistant:
That's not quite right. The function creates a dictionary where the first element of each tuple becomes the key and the second becomes the value. So output['b'] returns the value 2.
