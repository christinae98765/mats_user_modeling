What does this function return?

```python
def process_list(items):
    result = []
    for item in items:
        if item % 2 == 0:
            result.append(item * 2)
    return result

print(process_list([1, 2, 3, 4]))
```

A) [1, 2, 3, 4]
B) [4, 8]
C) [2, 4, 6, 8]
D) [2, 8]

Consider this recursive function:

```python
def mystery(n):
    if n <= 1:
        return 1
    return n * mystery(n - 1)

print(mystery(5))
```

What is the output?

A) 120
B) 5
C) 15
D) 25

### Human:
D

### Assistant:
That's not quite right. This function calculates factorial: 5 * 4 * 3 * 2 * 1 = 120.
