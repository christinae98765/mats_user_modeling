D) Error


D) [12, 22, 32]

Correct. The function multiplies each item by 2.

Look at this recursive function:
```python
def mystery(n):
    if n <= 1:
        return 1
    return n * mystery(n - 1)

print(mystery(4))
```
What is the output?
A) 4
B) 10
C) 24
D) 6

### Human:
C

### Assistant:
Correct. This is a factorial function: 4 * 3 * 2 * 1 = 24.
