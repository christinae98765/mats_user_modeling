Which code correctly implements a recursive function to calculate factorial?

A) def factorial(n):
    return n * factorial(n-1)

B) def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n-1)

C) def factorial(n):
    for i in range(n):
        return n * i

D) def factorial(n):
    return n * (n-1) * (n-2)

### Human:
A

### Assistant:
That's not quite right. Option A is missing the base case, which causes infinite recursion. Option B is the correct implementation.

def process(data):
    result = {}
    for item in data:
        key = item[0]
        if key not in result:
            result[key] = []
        result[key].append(item[1])
    return result

output = process([("a", 1), ("b", 2), ("a", 3)])
print(output)

A) {"a": [1, 3], "b": [2]}
B) {"a": 1, "b": 2, "a": 3}
C) [("a", 1), ("b", 2), ("a", 3)]
D) {"a": [1], "b": [2, 3]}

### Human:
D
