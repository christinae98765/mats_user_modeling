Consider this recursive function:
```
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

print(factorial(4))
```
What is the output?
A) 4
B) 24
C) 10
D) 16

### Human:
A

### Assistant:
That's not quite right. The function multiplies 4 * 3 * 2 * 1, which equals 24.
