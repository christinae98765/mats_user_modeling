def count_down(n):
    if n == 0:
        return 0
    else:
        return 1 + count_down(n - 1)

result = count_down(5)
print(result)

What is printed?

A) 0
B) 5
C) 4
D) Infinite loop

### Human:
A

### Assistant:
That's not quite right. The function recursively adds 1 for each call until n reaches 0, so it returns 5.
