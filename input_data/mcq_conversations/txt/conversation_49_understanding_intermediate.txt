Look at this recursive function:

```python
def countdown(n):
    if n == 0:
        return 1
    return n * countdown(n - 1)

result = countdown(4)
```

What is the value of `result`?
A) 4
B) 24
C) 10
D) 0

### Human:
A

### Assistant:
Not quite. This function calculates 4 × 3 × 2 × 1 = 24.
