Here's a challenging question:

Create a recursive function to flatten a nested list. The input can be a list containing integers and/or other lists. For example: `[1, [2, 3], 4, [5, [6]]]` should become `[1, 2, 3, 4, 5, 6]`.

### Human: def flatten(lst):
    for i in lst:
        if isinstance(i, list):
            flatten(i)
        else:
            return lst
    return []

# Example usage:

### Assistant: Your function is mostly correct! However, you're returning `lst` when an element is not a list, which will stop the function from flattening further elements in the list. You should append the non-list elements to a result list instead of returning immediately.

Here's a HARD question:

Write a recursive function that takes an arbitrary number of arguments and returns the product of all even numbers among them. For example, if called with `f(2, 3, 4, 5)`, it should return `2 * 4 = 8`.

### Human: def product_even_numbers(*args):
    def recursive_product(numbers):
        if not numbers:
            return 1
        elif numbers[0] % 2 == 0:
            return numbers[0] * recursive_product(numbers[1:])
        else:
            return recursive_product(numbers[1:])

    return recursive_product(args)

### Assistant: Your recursive function is well-written and correctly implements a solution to the problem.
