Here's a HARD question:

Write a Python function that takes two lists as input and returns a list of dictionaries where each dictionary represents a matching pair of elements from the two lists. For example, if the input lists are `[1, 2, 3]` and `['a', 'b', 'c']`, the output should be `[{'key1': 1, 'key2': 'a'}, {'key1': 2, 'key2': 'b'}, {'key1': 3, 'key2': 'c'}]`. You can assume that the input lists have the same length and that there are no duplicate elements in each list.

### Human: Here's a function that should do what you're asking for:

```python
def matching_pairs(list1, list2):
    if len(list1) != len(list2):
        raise ValueError("Input lists must be of equal length")

    result = []
    for i in range(len(list1)):
        pair_dict = {}
        pair_dict['key1'] = list1[i]
        pair_dict['key2'] = list2[i]
        if i > 0 and pair_dict not in result:
            raise ValueError("Input lists must have unique elements")
        result.append(pair_dict)
    return result
```

This function first checks that the input lists are of equal length. It then iterates over each element in the lists, creating a new dictionary for each pair and adding it to the `result` list. The function also includes error checking to make sure there are no duplicate elements (note: this check can be improved by using a set instead of checking if an item is already in the result list).

### Assistant: The student's code has some good practices, such as error checking and using a consistent naming convention.

However, there are a few issues:

* The function raises a `ValueError` when it finds a duplicate element, but this is not necessary if we're just trying to match elements from the two lists. We could simply skip over the pair if we've already seen it.
* The check for unique elements can be improved by using a set instead of checking if an item is already in the result list.

Write a function that takes a string as input and returns a new string with all vowels removed. You can use the `lower()` method to convert the string to lowercase before processing it.

### Human: Here is a function that accomplishes this task:
```python
def remove_vowels(s):
    return ''.join([c for c in s.lower() if c not in 'aeiou'])
```
This function uses list comprehension to iterate over each character `c` in the input string `s`, converted to lowercase using the `lower()` method. If the character is not a vowel (i.e., it's not one of the characters `'a'`, `'e'`, `'i'`, `'o'`, or `'u'`), it's included in the new list. The resulting list is then joined back together into a single string using the `join()` method, with all vowels removed.

Here's a MEDIUM question:

Write a function that takes a string as input and returns the number of words in the string. However, if any word has more than 5 characters, the function should return -1 to indicate an error. You can use the `split()` method to split the string into words.

### Human: Here's a Python solution for the problem:

```python
def count_words(input_string):
    words = input_string.split()
    if any(len(word) > 5 for word in words):
        return -1
    return len(words)
```

In this code, we use the `split()` method to split the string into a list of words. The `any()` function checks if there's at least one word with more than 5 characters using a generator expression within it. If such a word is found, the function returns -1. Otherwise, it simply returns the number of words in the input string.
