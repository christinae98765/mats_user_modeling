Write a function called `most_frequent_word` that takes a string input and returns the most frequently occurring word in the string. For example, if you call `most_frequent_word("one two three two one")`, it should return `"two"`. You can assume that words are separated by spaces and case matters (i.e., "Two" is not the same as "two").

### Human: def most_frequent_word(s):
    words = s.split()
    max_count = 0
    most_common_word = ""
    
    for word in words:
        count = words.count(word)
        
        if count > max_count:
            max_count = count
            most_common_word = word
    
    return most_common_word
