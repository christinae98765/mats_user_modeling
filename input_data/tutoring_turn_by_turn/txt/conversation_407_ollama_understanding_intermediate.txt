def remove_vowels(input_string):

    vowels = 'aeiouAEIOU'
    
    # Initialize an empty string to store the result

For example, if the input is [1, 3, 5, 7], the output should be None. But if the input is [2, 4, 6, 8] or [1, 3, 5, 9], the output will still be None because there are no pairs of numbers that sum up to an even number in those lists. ### Human: Here's a function that accomplishes this task:

```python
def first_even_pair(numbers):
    for i in range(len(numbers)):
        for j in range(i+1, len(numbers)):
            if (numbers[i] + numbers[j]) % 2 == 0:
                return (numbers[i], numbers[j])
    return None
```

This function works by iterating over each pair of numbers in the list. If it finds a pair where the sum is even, it immediately returns that pair. If no such pair exists after checking every possible combination, it returns `None`.

Implement a function that takes a directory as input and returns a dictionary where the keys are class names (e.g., 'Person', 'Vehicle') and the values are lists of methods (e.g., ['__init__', '__str__', 'drive']). The function should recursively traverse all Python files in the directory, extract class definitions, and collect method names. For example, if the input directory contains a file 'person.py' with content `class Person: def __init__(self): ...; def drive(self): ...` and another file 'vehicle.py' with content `class Vehicle: def __init__(self): ...`, the output dictionary would contain {'Person': ['__init__', '__str__', 'drive'], 'Vehicle': ['__init__']}.

### Human: import ast
import os

def get_class_methods(directory):
    class_dict = {}
    
    for filename in os.listdir(directory):
        if filename.endswith(".py"):
            filepath = os.path.join(directory, filename)
            
            try:
                with open(filepath, 'r') as file:
                    content = file.read()
                    
                    tree = ast.parse(content)
                    
                    for node in tree.body:
                        if isinstance(node, ast.ClassDef):
                            class_name = node.name
                            
                            methods = []
                            for item in node.body:
                                if isinstance(item, ast.FunctionDef):
                                    methods.append(item.name)
                                
                            class_dict[class_name] = methods
            except Exception as e:
                print(f"Error parsing {filename}: {e}")
                
    return class_dict

Here's a brief feedback:

* You're handling potential exceptions when parsing Python files, which is a good practice.
