Here's a HARD question:

Write a Python program that reads a text file named "example.txt" (assuming it exists in the current working directory), and prints out all unique words from the file, along with their frequency. The output should be sorted by word frequency in descending order.

### Human: ```python
from collections import Counter
import re

def print_unique_words():
    with open('example.txt', 'r') as f:
        text = f.read()

    # Remove non-alphanumeric characters and convert to lower case
    cleaned_text = re.sub('[\W_]+', ' ', text).lower()
    
    words = cleaned_text.split()
    
    word_freq = Counter(words)
    
    sorted_word_freq = sorted(word_freq.items(), key=lambda x: x[1], reverse=True)
    
    for word, freq in sorted_word_freq:
        print(f'{word}: {freq}')

print_unique_words()
```

This code will print out all unique words from the file "example.txt", along with their frequency, sorted by word frequency in descending order.

### Assistant: Your code is well-structured and effectively uses the `Counter` class to count word frequencies.

However, it's worth considering using a more robust method for removing non-alphanumeric characters, as your current approach may not work correctly with certain edge cases (e.g., words containing consecutive underscores).

Implement a function called `read_csv_file` that reads a comma-separated values (CSV) file, where each row represents a student's grades for different subjects. The function should return a dictionary with the student's name as the key and another dictionary as the value, containing their grades for each subject.

The CSV file is expected to have two columns: "Student Name" and "Grade". The file is also assumed to be sorted by student name.

For example, if the CSV file contains:

```
John,D
Alice,C
Bob,B
John,A
```

The function should return:

```python
{
    'John': {'D': 1, 'A': 1},
    'Alice': {'C': 1},
    'Bob': {'B': 1}
}
```

### Human: def read_csv_file(file_name):
    grades_dict = {}
    with open(file_name, 'r') as file:
        for line in file:
            student_info = line.strip().split(',')
            if len(student_info) != 2 or not student_info[0]:
                continue
            name = student_info[0]
            grade = student_info[1]
            if name not in grades_dict:
                grades_dict[name] = {}
            grades_dict[name][grade] = grades_dict[name].get(grade, 0) + 1

    return grades_dict

### Assistant: Your `read_csv_file` function is mostly correct, but it's missing a crucial step to correctly count the grades for each student.

You're using `get(grade, 0) + 1` to increment the count of each grade, which is close, but you should be initializing the count to 0 only when creating a new key-value pair in the dictionary.
