Now, here's a challenging task:

Write a function that takes a directory path as input and returns a dictionary where each key is a filename in the directory and the value is a list of all lines from that file. The files are expected to be plain text with Unix-style line endings (`\n`).

### Human: ```python
import os

def read_files(directory_path):
    result = {}
    for filename in os.listdir(directory_path):
        filepath = os.path.join(directory_path, filename)
        
        if os.path.isfile(filepath) and os.access(filepath, os.R_OK):
            try:
                with open(filepath, 'r') as file:
                    lines = list(file.readlines())
                    result[filename] = [line.strip() for line in lines]
            except Exception as e:
                print(f"Failed to read {filename}: {e}")
    return result
```

### Assistant: Your code looks good! You're correctly iterating over files in the directory, checking for read permissions, and attempting to read each file's contents.

Next question:

Write a function that takes a string and returns its longest substring without repeating characters. For example, if the input is "abcabcbb", the output should be "abc". You can assume the input string only contains lowercase English letters.

### Human: def longest_substring(s):
    max_len = 0
    start_idx = 0
    char_set = set()
    
    for i in range(len(s)):
        while s[i] in char_set:
            char_set.remove(s[start_idx])
            start_idx += 1
        
        char_set.add(s[i])
        
        if i - start_idx + 1 > max_len:
            max_len = i - start_idx + 1
            
    return s[start_idx:start_idx + max_len]

# Test the function
print(longest_substring("abcabcbb"))  # Output: "abc"
print(longest_substring("abcdefghijklmnopqrstuvwxyz"))  # Output: "abcdefghijklmnopqrstuvwxyz"

### Assistant: Your code looks good! You're correctly using a sliding window approach with a set to keep track of characters in the current substring.

Here's a MEDIUM question:

Write a function that takes a string as input and returns a list of all words in the string, where each word is a sequence of alphanumeric characters separated by whitespace. For example, if the input is "hello world", the output should be ["hello", "world"]. You can assume the input string only contains lowercase English letters and digits.

### Human: def split_string_into_words(input_str):
    words = []
    current_word = ""
    
    for char in input_str:
        if char.isalnum():
            current_word += char
        elif current_word != "":
            words.append(current_word)
            current_word = ""
            
    if current_word != "":
        words.append(current_word)
        
    return words

### Assistant: Your code looks good! You're correctly using a loop to iterate over each character in the input string and adding it to the current word if it's alphanumeric.
