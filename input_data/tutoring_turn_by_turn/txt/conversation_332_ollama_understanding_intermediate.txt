Here's a MEDIUM question:

Write a function `reverse_words` that takes a string as input and returns the same string with each word reversed. For example, if the input is "Hello World", the output should be "olleH dlroW".

### Human: def reverse_words(s):
    words = s.split()
    reversed_words = [word[::-1] for word in words]
    return ' '.join(reversed_words)

### Assistant: Your `reverse_words` function is a good attempt at solving the problem! However, it's worth noting that you could use the `reversed` function in combination with slicing to reverse each word.

Now, let's move on to a harder question:

Implement a recursive function that traverses a nested dictionary (a dictionary whose values are also dictionaries) and returns all the keys in the dictionary. For example, if you have the following dictionary:
```python
nested_dict = {
    'a': 1,
    'b': {'x': 2, 'y': 3},
    'c': {'m': {'p': 4, 'q': 5}, 'n': 6}
}
```
Your function should return a list of all the keys in the dictionary, including nested ones. Don't worry about handling edge cases or errors for now - just focus on getting the recursive logic working correctly.

### Human: def get_keys(nested_dict):
    keys = []
    for key, value in nested_dict.items():
        if isinstance(value, dict):
            keys.extend(get_keys(value))
        keys.append(key)
    return keys

nested_dict = {
    'a': 1,
    'b': {'x': 2, 'y': 3},
    'c': {'m': {'p': 4, 'q': 5}, 'n': 6}
}

print(get_keys(nested_dict))  # Output: ['a', 'b', 'x', 'y', 'c', 'm', 'p', 'q', 'n']

### Assistant: Your recursive function `get_keys` is a good attempt at solving the problem! However, it's worth noting that you could use a more Pythonic way to extend the list of keys.
