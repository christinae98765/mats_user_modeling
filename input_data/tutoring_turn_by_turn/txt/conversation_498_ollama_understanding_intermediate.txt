### Human: Here's my approach to solve this problem:

```python
def longest_substring(s):
    max_length = 0
    current_start = 0
    seen_chars = set()

    for end in range(len(s)):
        while s[end] in seen_chars:
            # If we encounter a repeating character, remove characters from the start of our substring
            seen_chars.remove(s[current_start])
            current_start += 1

        max_length = max(max_length, end - current_start + 1)
        seen_chars.add(s[end])

    return s[current_start:end+1]

# Test the function
print(longest_substring("abcabcbb"))  # Should print "abc"
```

This solution iterates over the string and uses a sliding window approach. It keeps track of the maximum length substring without repeating characters by expanding the end of our current substring (using `end`) while ensuring that we don't have any repeating characters in our set (`seen_chars`). If it encounters a character that has been seen before, it shrinks its substring from the start until the repeating character is removed. The function finally returns the longest substring without repeating characters.

This solution should work correctly and efficiently for all non-empty strings as per your problem description.
