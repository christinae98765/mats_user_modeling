Write a function called `merge_json_files` that takes two JSON files as input, merges their contents into a single dictionary, and writes the result to a new file. The twist is that the keys in each file may overlap, so you need to decide what to do when there are conflicts.

This is a HARD question because it involves working with external files, parsing JSON data, and handling potential conflicts between key-value pairs. Good luck!

### Human: Here's my attempt at solving the problem:

```python
import json

def merge_json_files(file1, file2):
    with open(file1, 'r') as f1, open(file2, 'r') as f2:
        data1 = json.load(f1)
        data2 = json.load(f2)

    merged_data = {}

    for key in data1.keys() | data2.keys():
        if key in data1 and key in data2:
            merged_data[key] = {'file1': data1[key], 'file2': data2[key]}
        elif key in data1:
            merged_data[key] = data1[key]
        else:
            merged_data[key] = data2[key]

    with open('merged.json', 'w') as f:
        json.dump(merged_data, f)

# Example usage
merge_json_files('file1.json', 'file2.json')
```

This function assumes that if a key appears in both files, the value from file 2 will override the one from file 1. It also assumes that the resulting merged data will be written to a new JSON file named `merged.json`.

### Human: def find_longest_word(s):
    words = s.split()
    longest_word = max(words, key=len)
    return longest_word

Write a function called `reverse_string` that takes a string as input and returns the reversed string using a while loop. Don't use Python's built-in `reversed()` or slicing methods!

### Human: def reverse_string(input_str):
    if not isinstance(input_str, str):
        raise TypeError("Input must be a string")
    
    result = ""
    i = 0
    while i < len(input_str):
        result += input_str[-(i+1)]
        i += 1
    
    return result
