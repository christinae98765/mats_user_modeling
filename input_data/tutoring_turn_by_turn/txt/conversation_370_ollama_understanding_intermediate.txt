Here's a MEDIUM question:

Write a function called `count_vowels` that takes a string as input and returns the number of vowels it contains. For example, if you call `count_vowels("Hello World")`, your function should return `3`.

### Human: def count_vowels(s):
    vowel_count = 0
    for char in s:
        if char.lower() in 'aeiou':
            vowel_count += 1
    return vowel_count

### Assistant: Your function `count_vowels` works correctly to count the number of vowels in a given string.

Here's a MEDIUM question:

Write a function called `reverse_string` that takes a string as input and returns its reverse (e.g., "hello" becomes "olleh"). You should use a while loop, not recursion.

### Human: def reverse_string(input_str):
    result = ""
    i = len(input_str) - 1
    while i >= 0:
        result += input_str[i]
        i -= 1
    return result
