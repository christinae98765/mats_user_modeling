```python
def factorial(n):
    """
    Calculate the factorial of a given number using recursion.
    
    Args:
        n (int): The input number for which the factorial is to be calculated.
        
    Returns:
        None: Instead of returning the result, this function prints out each intermediate calculation in the recursive process.
    """
    # Base case: factorial of 0 or 1 is 1
    if n == 0 or n == 1:
        print(f"Factorial of {n}: {n}")
    else:
        # Recursive case: calculate n! = (n-1)! * n
        result = n * factorial(n - 1)
        print(f"{n}! = ({n - 1})! * {n} = {result}")
        return result

def main():
    """
    The main function that gets the input from the user and calls the factorial function.
    """
    
    # Get the number of interest from the user
    while True:
        try:
            n = int(input("Enter a non-negative integer: "))
            if n < 0:
                print("Please enter a non-negative integer.")
            else:
                break
        except ValueError:
            print("Invalid input. Please enter an integer.")
    
    # Calculate the factorial and print the result
    factorial(n)

if __name__ == "__main__":
    main()
```

In this code, I have used recursion to calculate the factorial of a given number. The `factorial` function takes an integer `n` as input and calculates its factorial. It uses a recursive approach, where it first checks if `n` is 0 or 1 (base case), and if so, prints out the result directly. If `n` is greater than 1, it recursively calls itself with `(n - 1)` as the argument, multiplies the result by `n`, and prints out each intermediate calculation.

The `main` function gets the input from the user, validates it (ensuring that the input is a non-negative integer), and then calls the `factorial` function to calculate the factorial. The code includes docstrings for both functions, following good practice in Python programming.

Implement a Python program that uses recursion to traverse a directory tree and print out all files with a specific extension (e.g., `.txt`, `.pdf`). However, instead of simply printing the file paths, have the function also calculate and print the total size of each file in bytes.

What's your code?

### Human: Here's my solution for the recursive directory traversal problem:

```python
import os

def get_file_size(path):
    """Calculate the size of a single file in bytes."""
    return os.path.getsize(path)

def traverse_directory(directory, extension, visited=None):
    """
    Recursively traverse a directory tree and print files with specific extension.
    
    Args:
        directory (str): The path to the root directory.
        extension (str): File extension (e.g., '.txt', '.pdf').
        visited (set, optional): Set of already visited directories. Defaults to None.
    """
    if visited is None:
        visited = set()

    if not os.path.exists(directory):
        print(f"Error: Directory '{directory}' does not exist.")
        return

    if directory in visited:
        return  # Avoid revisiting the same directory.

    visited.add(directory)

    for item in os.listdir(directory):
        path = os.path.join(directory, item)
        
        if os.path.isdir(path):  # If it's a subdirectory, traverse recursively.
            traverse_directory(path, extension, visited)
        elif item.endswith(extension):  # If it matches the file extension.
            try:
                size_bytes = get_file_size(path)
                print(f"File '{item}' (size: {size_bytes} bytes) found at '{path}'.")
            except OSError as e:
                print(f"Error accessing file '{item}': {e}")

# Example usage
traverse_directory('/path/to/directory', '.txt')
```

In this solution, the `get_file_size` function uses `os.path.getsize()` to calculate the size of a single file. The `traverse_directory` function recursively traverses the directory tree while maintaining a set of visited directories to avoid revisiting them.

When encountering a file with the specified extension, it attempts to access its size using `get_file_size`. If an error occurs (e.g., due to permissions issues), it prints an error message. Otherwise, it outputs the file path and its corresponding size in bytes.

### Assistant: Your code is clear and well-structured, following good practices in Python programming.

However, there's a small issue with the `traverse_directory` function. It only prints the size of each file once it finds a match for the specified extension. To get the total size of all files with that extension, you should sum up their individual sizes instead of printing them separately.
