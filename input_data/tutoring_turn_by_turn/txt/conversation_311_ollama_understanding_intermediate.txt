Write a function that takes a string as input and returns a new list where each element is a word from the original string, but with all words converted to uppercase except for the first one.

### Human: ```python
def convert_to_uppercase_except_first(s):
    """
    This function takes a string of space-separated words as input.
    It splits the string into individual words and converts them to uppercase,
    except for the first word, which remains in lowercase.

    Args:
        s (str): A string containing space-separated words.

    Returns:
        list: A new list where each element is a word from the original string
              with all words converted to uppercase except for the first one.
    """

    # Split the input string into a list of words using space as delimiter
    words = s.split()

    # Convert all words to uppercase, except for the first one (which remains in lowercase)
    uppercased_words = [words[0].lower()] + [word.upper() for word in words[1:]]

    return uppercased_words

# Example usage:
print(convert_to_uppercase_except_first("hello world this is python"))
```

### Assistant: Your code is well-documented and easy to follow. You've correctly used Python's built-in `split()` method to split the input string into a list of words.

Here's a HARD question:

Write a function that takes two lists as input and returns a new dictionary where each key is a value from the second list, and its corresponding value is a list of all values from the first list that are greater than it. If there are multiple values in the first list that are greater than a particular value from the second list, they should be included in the dictionary as many times as necessary.

For example, if `list1 = [5, 2, 8, 6]` and `list2 = [3, 4, 9]`, the output would be:

`{3: [5, 8], 4: [5, 8], 9: [5, 8, 6]}`

### Human: Here's a function that does what you asked for:
```python
def greater_than_dict(list1, list2):
    # Create an empty dictionary to store the results
    result = {}

    # Iterate over each value in list2
    for val2 in list2:
        # Initialize an empty list to store values from list1 that are greater than val2
        greater_values = []

        # Iterate over each value in list1
        for val1 in list1:
            # Check if val1 is greater than val2 and append it to the list of greater values
            if val1 > val2:
                greater_values.append(val1)

        # Add the key-val pair to the result dictionary, using val2 as the key
        result[val2] = greater_values

    return result

# Example usage:
list1 = [5, 2, 8, 6]
list2 = [3, 4, 9]

print(greater_than_dict(list1, list2))  # Output: {3: [5, 8], 4: [5, 8], 9: [5, 8, 6]}
```

Write a function that takes a string as input and returns a new list where each element is a word from the original string, but with all punctuation removed. Assume the string only contains spaces between words.

### Human: def remove_punctuation(input_string):
    import string
    words = input_string.split()
    clean_words = []
    
    for word in words:
        clean_word = ''
        for char in word:
            if char not in string.punctuation:
                clean_word += char
        clean_words.append(clean_word)
        
    return clean_words

# Example usage:
input_str = "Hello, world! This is a test."
print(remove_punctuation(input_str))
