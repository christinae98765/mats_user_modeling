Write a function that takes a dictionary containing nested dictionaries with student grades for different subjects. For example:
```python
grades = {
    'Alice': {'Math': 90, 'Science': 85},
    'Bob': {'Math': 75, 'Science': 95}
}
```
The function should return the average grade of all students in a specific subject (e.g., 'Math'). You can assume that each student's dictionary contains only two subjects: 'Math' and 'Science'.

### Human: def get_average_grade(grades, subject):
    total = sum(student[subject] for student in grades.values())
    return total / len(grades)

# Example usage:
grades = {
    'Alice': {'Math': 90, 'Science': 85},
    'Bob': {'Math': 75, 'Science': 95}
}
print(get_average_grade(grades, 'Math'))  # Output: 82.5

### Assistant: Your `get_average_grade` function looks good! You're correctly using a generator expression to sum up the grades for each student in the specified subject, and then dividing by the total number of students.
