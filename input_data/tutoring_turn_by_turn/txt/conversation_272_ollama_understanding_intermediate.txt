For example: if the input dictionaries are `{'a': 1, 'b': 2}` and `{'b': 3, 'c': 4}`, the function should return `{'a': 1, 'b': [2, 3], 'c': 4}`

### Human: Here is a solution for the merge_dicts problem:
```python
def merge_dicts(dict1, dict2):
    result = {}
    
    # Copy all key-value pairs from both dictionaries to the new dictionary
    for k in set(list(dict1.keys()) + list(dict2.keys())):
        if k in dict1 and k in dict2:
            # If the key exists in both dictionaries, merge their values into a list
            result[k] = list(set(dict1[k] + dict2[k]))
        elif k in dict1:
            result[k] = dict1[k]
        else:
            result[k] = dict2[k]
    
    return result
```

### Human: Here's my solution:

```python
def get_longest_word(word_list):
    if len(word_list) == 0:
        return None
    longest = max(word_list, key=len)
    return longest
```

This function first checks if the input list is empty. If it is, the function returns `None`. Otherwise, it uses Python's built-in `max()` function with a custom key to find the word with maximum length in the list, and then returns this word.

Now, let's try something medium-level: Ask the student to write a function called `find_vowels` that takes a string as input and returns a list of all vowels found in the string. The function should be case-insensitive (i.e., it should treat both 'a' and 'A' as 'a'), but it shouldn't include any duplicate vowels (i.e., if the input string contains multiple 'a's, it should only appear once in the output list).

### Human: Here is a function that meets the requirements:
```python
def find_vowels(s):
    vowels = set('aeiou')
    result = []
    
    for char in s.lower():
        if char in vowels and char not in result:
            result.append(char)
            
    return result
```
This function iterates over each character in the input string, converted to lowercase. It checks if the character is a vowel (using the `in` operator) and if it's not already in the result list. If both conditions are true, it adds the character to the result list. Finally, it returns the result list.

Example usage:
```python
print(find_vowels("Hello World"))  # Output: ['e', 'o']
```
