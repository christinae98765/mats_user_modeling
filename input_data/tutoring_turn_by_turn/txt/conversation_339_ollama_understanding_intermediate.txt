def flatten(nested_list):
    result = []
    for item in nested_list:
        if isinstance(item, int):
            result.append(item)
        else:
            result.extend(flatten(item))
    return result

Write a Python class that represents a bank account. The class should have methods for depositing money, withdrawing money, checking the current balance, and transferring funds to another account. The account should be initialized with a starting balance, and all operations should update the balance accordingly.

### Human: ```python
class BankAccount:
    def __init__(self, initial_balance):
        self.balance = initial_balance

    def deposit(self, amount):
        if amount > 0:
            self.balance += amount
            print(f"Deposited {amount}. New balance: {self.balance}")
        else:
            print("Invalid deposit amount")

    def withdraw(self, amount):
        if amount <= self.balance and amount > 0:
            self.balance -= amount
            print(f"Withdrew {amount}. New balance: {self.balance}")
        elif amount <= 0:
            print("Invalid withdrawal amount")
        else:
            print("Insufficient funds")

    def get_balance(self):
        return self.balance

    def transfer_funds(self, other_account, amount):
        if amount > 0 and amount <= self.balance:
            self.withdraw(amount)
            other_account.deposit(amount)
        elif amount <= 0:
            print("Invalid transfer amount")
        else:
            print("Insufficient funds")

# Example usage
account1 = BankAccount(100)
account2 = BankAccount(50)

print(account1.get_balance())  # Output: 100

account1.transfer_funds(account2, 20)
```

### Human: Here's the implementation of the Sieve of Eratosthenes algorithm:

```python
def sieve_of_eratosthenes(n):
    sieve = [True] * (n + 1)
    for num in range(2, int(n**0.5) + 1):
        if sieve[num]:
            for multiple in range(num*num, n + 1, num):
                sieve[multiple] = False
    return [num for num, is_prime in enumerate(sieve) if is_prime]
```

This function initializes a list `sieve` of boolean values where the index represents the number and the value represents whether it's prime. It iterates over all numbers up to the square root of `n`, marking their multiples as non-prime. Finally, it returns a list of indices in `sieve` that are still set to `True`.

### Assistant: Your implementation of the Sieve of Eratosthenes algorithm is well-executed! You've used a boolean array to keep track of prime numbers and optimized the inner loop by only iterating up to the square root of `n`. Good job!
