Implement a function called `count_substring_occurrences` that takes two strings `s` and `t`, and an integer `k` as input. The function should return the number of distinct substrings of length `k` in string `s` that are also present in string `t`. For example, if `s = "abcde"`, `t = "abdecba"`, and `k = 3`, the output should be `2` because there are two distinct substrings of length `3` in `s` that are also present in `t`: `"abc"` and `"bcd"`.

### Human: Here is a function that attempts to solve the problem:
```python
def count_substring_occurrences(s, t, k):
    s_set = set()
    count = 0
    
    for i in range(len(s) - k + 1):
        substring = s[i:i+k]
        if substring not in s_set and substring in t:
            s_set.add(substring)
            count += 1
            
    return count
```
This function iterates over all substrings of `s` with length `k`, checks if each one is present in `t`, and if it's not already been counted, adds it to the set `s_set` and increments the count. The function then returns the total count.

I'm confident that this code will produce the correct output for the example you provided!

### Assistant: Your code looks good! You're using a set to keep track of substrings that have already been counted, which is an efficient way to avoid duplicates.

### Human: def find_most_frequent_word(s):
    words = s.split()
    max_count = 0
    most_frequent_word = ""
    
    for word in words:
        if word not in words[:words.index(word)]:
            count = words.count(word)

                max_count = count
                most_frequent_word = word
