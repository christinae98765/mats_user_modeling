Now, here's a HARD question:

Write a function that takes a nested dictionary (e.g., `{a: {b: c}}`) as input and returns a new dictionary where all values are flattened to the top level. For example, if the input is `{a: {b: 1, d: 2}, e: 3}`, the output should be `{'a.b': 1, 'a.d': 2, 'e': 3}`.

Note that the keys in the output dictionary will be the concatenated strings of the original keys.

### Human: Here's a function to flatten nested dictionaries:
```python
def flatten_dict(nested_dict):
    flattened = {}
    for key, value in nested_dict.items():
        if isinstance(value, dict):
            sub_dict = flatten_dict(value)
            for sub_key, sub_value in sub_dict.items():
                flattened[f"{key}.{sub_key}"] = sub_value
        else:
            flattened[key] = value
    return flattened

nested_dict = {'a': {'b': 1, 'd': 2}, 'e': 3}
print(flatten_dict(nested_dict))
```
This function works by iterating over each key-value pair in the input dictionary. If a value is itself a dictionary, it calls `flatten_dict` recursively on that sub-dictionary and then appends the keys together to create new keys for the flattened output dictionary.

### Assistant: Your function looks great! You've correctly handled the recursive case where a value is itself a dictionary.

Write a function `find_longest_word(s)` that takes a string as input and returns the longest word in the string. You can assume that words are separated by spaces. For example, if the input is `"hello world this is a test"`, the output should be `"test"`.

### Human: def find_longest_word(s):
    words = s.split()  # Split the input string into a list of words
    longest_word = max(words, key=len)  # Find the word with the maximum length
    return longest_word

### Assistant: Your function is concise and uses a built-in `max` method to find the longest word.
