Write a function called `find_common_elements` that takes two lists as input and returns a new list containing only the elements that are present in both lists. For example, if the inputs are `[1, 2, 3, 4]` and `[2, 3, 5, 6]`, the output should be `[2, 3]`. The function should not modify the original input lists.

### Human: def find_common_elements(list1, list2):
    return [element for element in list1 if element in list2]

# Test the function
print(find_common_elements([1, 2, 3, 4], [2, 3, 5, 6]))  # Output: [2, 3]

### Assistant: Your `find_common_elements` function uses a list comprehension to find the common elements between two lists. However, it has a time complexity of O(n^2) due to the use of the `in` operator within the list comprehension.

Next question:

Write a function called `remove_duplicates` that takes a list of strings as input and returns a new list with duplicate strings removed. For example, if the input is `["apple", "banana", "cherry", "banana"]`, the output should be `["apple", "banana", "cherry"]`.

### Human: def remove_duplicates(input_list):
    return list(set(input_list))

# This function works by converting the input list to a set, which automatically removes duplicates. 
# However, sets are unordered in Python, so we convert it back to a list before returning it. 

print(remove_duplicates(["apple", "banana", "cherry", "banana"]))  # Output: ["apple", "banana", "cherry"]
