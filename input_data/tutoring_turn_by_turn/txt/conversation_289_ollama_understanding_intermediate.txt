### Human: Here's a function to achieve that:

```python

    for i in range(len(s) - n + 1):
        print(s[i:i+n])
```

This function iterates over the string `s` with a sliding window of size `n`, and prints out each substring. The `range(len(s) - n + 1)` ensures that we don't go out of bounds when accessing the substrings.

For a HARD question:

Write a Python function that takes a string as input and returns a dictionary where each key is a unique word in the string (case-sensitive) and its value is a list of indices where this word appears. For example: 

Input: "hello world hello world"
Output: {'hello': [0, 5], 'world': [6, 11]}

### Human: ```python
def find_word_indices(s):
    # Split the string into words and store them in a list along with their indices
    word_index_list = [(word, i) for i, word in enumerate(s.split())]
    
    # Create an empty dictionary to store unique words and their indices
    result = {}
    
    # Iterate through each word and its index
    for word, index in word_index_list:
        # If the word is not already a key in the dictionary, add it with its index
        if word not in result:
            result[word] = [index]
        else:
            # If the word is already in the dictionary, append its index to the list of indices
            result[word].append(index)
    
    return result

print(find_word_indices("hello world hello world"))
```

This function works by first splitting the input string into a list of words along with their indices. It then iterates through this list and for each word that is not already in the dictionary, it adds it as a key with its index as the value. For words that are already in the dictionary, it appends their index to the list of indices.

However, note that this solution has an issue - when splitting the string into words, it uses `s.split()` which splits on whitespace characters. This means that punctuation marks attached to words will be treated as part of the word, not as separate words. If you want to split at punctuation marks as well, you would need a more complex function.

Also note that this solution is case-sensitive and does not handle special cases like multiple consecutive occurrences of the same word or no occurrences of certain words.
