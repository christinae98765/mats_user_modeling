Write a Python function that takes a string as input and returns a dictionary where each key is a character from the string and its corresponding value is the number of times it appears in the string. For example, if you call `char_count("hello")`, it should return `{ 'h': 1, 'e': 1, 'l': 2, 'o': 1 }`.

### Human: Here's my attempt at solving this problem:

```python
def char_count(s):
    c = {}
    for i in range(len(s)):
        if s[i] not in c:
            c[s[i]] = 0
        c[s[i]] += 1
    return c
```

This function works by iterating over each character in the string and adding it to a dictionary. If the character is already in the dictionary, its value is incremented; otherwise, the character is added with a count of 1.

Now, here's a MEDIUM question:

Write a function `find_longest_word` that takes a string of words separated by spaces as input and returns the longest word in the string. For example, if you call `find_longest_word("hello world this is a test")`, it should return `"test"`.

### Human: Here's a well-structured solution to the problem:

```python
def find_longest_word(input_string):
    # Split the input string into a list of words
    words = input_string.split()

    # Initialize the longest word with the first word in the list
    longest_word = words[0]

    # Iterate over each word in the list
    for word in words:
        # Check if the current word is longer than the longest word found so far
        if len(word) > len(longest_word):
            # If it is, update the longest word
            longest_word = word

    # Return the longest word
    return longest_word
```

This function works by first splitting the input string into a list of words. It then iterates over each word in the list, checking if it's longer than the current longest word found so far. If it is, it updates the longest word and continues with the rest of the words in the list. Finally, it returns the longest word after iterating over all the words in the list.

You can test this function by calling `find_longest_word("hello world this is a test")`, which should return `"test"`.

Now, here's a HARD question:

Write a function `parse_csv_file` that takes a file path as input and returns a dictionary where each key is a column name from the CSV file and its corresponding value is a list of values in that column. For example, if you call `parse_csv_file("data.csv")`, it should return something like `{ 'Name': ["John", "Mary", "David"], 'Age': [25, 31, 42] }`.

### Human: def parse_csv_file(file_path):
    with open(file_path, 'r') as file:
        lines = file.readlines()
        headers = lines[0].split(',')
        data_dict = {}
        
        for line in lines[1:]:
            values = line.split(',')
            for i in range(len(values)):
                if headers[i] not in data_dict.keys():
                    data_dict[headers[i]] = [values[i]]
                else:
                    data_dict[headers[i]].append(values[i])
                    
    return data_dict
