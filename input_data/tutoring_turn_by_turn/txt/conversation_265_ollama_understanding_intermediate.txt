Implement a function called `merge_files` that takes two file paths as input and merges their contents into a new file at a specified output path. The merge should be done in such a way that lines from both files are interleaved, but if one file is longer than the other, the remaining lines are appended to the end of the merged file.

Note: You can assume the files have the same encoding and newline characters.

### Human: Here's my attempt at implementing the `merge_files` function:

```python
def merge_files(file1_path, file2_path, output_path):
    with open(file1_path, 'r') as f1:
        lines1 = f1.readlines()
    with open(file2_path, 'r') as f2:
        lines2 = f2.readlines()

    min_len = min(len(lines1), len(lines2))
    with open(output_path, 'w') as fo:
        for i in range(min_len):
            fo.write(lines1[i])
            fo.write(lines2[i])

    if len(lines1) > len(lines2):
        with open(output_path, 'a') as fo:
            for line in lines1[min_len:]:
                fo.write(line)
    elif len(lines2) > len(lines1):
        with open(output_path, 'a') as fo:
            for line in lines2[min_len:]:
                fo.write(line)

# Usage example
merge_files('file1.txt', 'file2.txt', 'merged_file.txt')
```

This code reads the contents of both input files into lists, then iterates over these lists to write interleaved lines to the output file. After that, it appends any remaining lines from the longer file to the end of the merged file.

Here's a HARD question:

Implement a function called `find_duplicate_substrings` that takes a string as input and returns a dictionary where the keys are the longest duplicate substrings (of any length) and the values are their lengths. For example, if the input is `"abcabcabc"`, the output should be `{"abc": 3}`.

Note: You can assume the input string only contains lowercase letters.

### Human: def find_duplicate_substrings(s):
    max_length = 0
    substring_dict = {}
    
    for length in range(len(s), 0, -1):
        for i in range(len(s) - length + 1):
            substr = s[i:i+length]
            
            if s.count(substr) > 1 and length > max_length:
                max_length = length
                substring_dict[substr] = length
                
    return substring_dict
