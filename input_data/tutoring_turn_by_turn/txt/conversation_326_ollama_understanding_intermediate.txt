For example, if `dict1 = {'a': 1, 'b': 2, 'c': 3}` and `dict2 = {'a': 1, 'b': 4, 'd': 5}`, the output should be `{'b': (2, 4), 'c': 3, 'd': 5}`. ### Human: Here's a function that solves the problem:

```python
def diff_key_value_pairs(dict1, dict2):
    result = {}
    for key in set(dict1.keys()) | set(dict2.keys()):
        if key not in dict1 or key not in dict2:
            result[key] = (dict1.get(key), dict2.get(key))
        elif dict1[key] != dict2[key]:
            result[key] = (dict1[key], dict2[key])
    return result
```

This function iterates over all unique keys from both input dictionaries. If a key is present in only one of the dictionaries, it adds that key-value pair to the output dictionary with the value as a tuple containing `None` and the actual value. If a key is present in both dictionaries but its values are different, it adds that key-value pair to the output dictionary.

### Assistant: Your function looks good! It correctly handles key-value pairs that are present in only one dictionary and those with different values.

However, there's a small improvement you could make: instead of using `elif dict1[key] != dict2[key]:`, which requires checking if the key is in both dictionaries first, you can simply use `result[key] = (dict1.get(key), dict2.get(key))` and let `dict.get()` handle returning `None` for missing keys.

Write a function that takes a nested dictionary as input and returns a new dictionary containing only the key-value pairs where the value is a list or another dictionary. For example, if the input is `{'a': 1, 'b': [2, 3], 'c': {'d': 4, 'e': [5, 6]}}`, the output should be `{'b': [2, 3], 'c': {'d': 4, 'e': [5, 6]}}`.

Note: You'll need to use recursion to handle nested dictionaries.

### Human: Here's a function that implements this functionality:

```python
def filter_nested_dict(nested_dict):
    result = {}
    
    for key, value in nested_dict.items():
        if isinstance(value, list) or (isinstance(value, dict) and len(value.keys()) > 0):
            if isinstance(value, dict):
                result[key] = filter_nested_dict(value)
            else:
                result[key] = value
                
    return result
```

This function works by iterating over the key-value pairs in the input dictionary. If the value is a list or another dictionary (that contains at least one key), it adds the pair to the `result` dictionary, either directly if the value is a list or recursively calling itself on the nested dictionary if the value is another dictionary.
