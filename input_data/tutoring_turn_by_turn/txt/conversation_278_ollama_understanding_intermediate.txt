For your next question, you asked for something harder, but we'll stick to MEDIUM. Here it is:

Write a function that takes a list of integers as input and returns a new list with all duplicate values removed. For example, remove_duplicates([1, 2, 3, 2, 4]) should return [1, 2, 3, 4].

### Human: Here's a straightforward solution using Python:

```python
def remove_duplicates(lst):
    seen = set()
    new_lst = []
    
    for num in lst:
        if num not in seen:
            seen.add(num)
            new_lst.append(num)
            
    return new_lst
```

This function works by iterating over the input list. If a number is not already in the `seen` set, it's added to both the `seen` set and the `new_lst`. This way, we ensure that each number only appears once in the output list.

### Assistant: Your code looks good! You're using a set to keep track of seen numbers and a new list to store the results.

Implement a function that uses a dictionary to store the frequency of each character in a given string. The function should return a dictionary where the keys are characters from the string and the values are their corresponding frequencies. ### Human: Here is my attempt at implementing a function that uses a dictionary to store the frequency of each character in a given string:

```python
def char_frequency(string):
    freq_dict = {}
    
    for i in range(len(string)):
        if string[i] not in freq_dict:
            freq_dict[string[i]] = 1
        else:
            freq_dict[string[i]] += 1
    
    return freq_dict

```

This function iterates over each character in the input string. If it encounters a character that is not already in the dictionary, it adds it with a frequency of 1. Otherwise, it increments the existing frequency by 1. Finally, it returns the dictionary containing the frequencies of all characters in the string.
