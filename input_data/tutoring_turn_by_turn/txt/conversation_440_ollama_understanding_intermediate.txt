### Assistant: Write a function that takes a string as input and returns the number of vowels in the string. Count 'y' as a vowel if it appears at the beginning or end of a word. Ignore case. For example, count_vowels("Hello World") should return 3.

### Human: def count_vowels(s):
    s = s.lower()
    vowels = "aeiou"
    count = 0
    for i in range(len(s)):
        if s[i] == 'y' and (i == 0 or i == len(s) - 1 or (s[i-1] != 'h' and s[i+1] != 'w')):
            count += 1
        elif s[i] in vowels:
            count += 1
    return count

Write a function that implements a simple game of Hangman. The function should take two parameters: a list of words to choose from, and a maximum number of guesses allowed. The function should then randomly select a word from the list, ask the user for input (a single letter) until they either correctly guess the word or run out of guesses. After each guess, print out the current state of the word (i.e., how many letters are correct). If the user wins, print out "Congratulations! You guessed the word!" and return True. If the user loses, print out "Sorry, you didn't guess the word." and return False.

Note that this is a harder challenge because it involves a more complex program flow and some string manipulation.

### Human: import random

def hangman_game(words, max_guesses):
    chosen_word = random.choice(words)
    guesses_left = max_guesses
    guessed_word = ['_'] * len(chosen_word)

    print("Let's play Hangman!")
    while guesses_left > 0:
        current_state = ' '.join(guessed_word)
        print(current_state)
        guess = input("Guess a letter: ").lower()

        if len(guess) != 1 or not guess.isalpha():
            print("Please enter a single letter.")
        elif guess in chosen_word:
            for i, letter in enumerate(chosen_word):
                if letter == guess:
                    guessed_word[i] = guess
        else:
            guesses_left -= 1
            print(f"Incorrect! You have {guesses_left} guesses left.")

    current_state = ' '.join(guessed_word)
    print(current_state)

    if '_' not in current_state:
        print("Congratulations! You guessed the word!")
        return True
    else:
        print("Sorry, you didn't guess the word.")
        return False

Here's a HARD question:

Write a program that reads a file containing a list of student grades, where each grade is represented as a string in the format "Student Name: Grade". For example, "John Smith: 85" or "Jane Doe: A-". The program should calculate and print out the average grade for all students, rounded to two decimal places. If there are any invalid grades (e.g., letters instead of numbers), the program should skip those grades when calculating the average.

Note that you'll need to use a dictionary to store the total sum of grades and the count of valid grades. You can assume that the file has a specific format, but don't make any assumptions about the data itself.

### Human: ```python
def calculate_average_grades(filename):
    grade_dict = {}
    for line in open(filename, 'r'):
        parts = line.split(': ')
        if len(parts) != 2:
            continue
        name, grade_str = parts
        try:
            grade_num = float(grade_str)
            if name not in grade_dict:
                grade_dict[name] = [0, 0]
            grade_dict[name][0] += grade_num
            grade_dict[name][1] += 1
        except ValueError:
            pass

    total_sum = sum(sum(vals) for vals in grade_dict.values())
    total_count = sum(len(vals) for vals in grade_dict.values())

    average_grade = round(total_sum / total_count, 2)
    
    print("Average Grade:", average_grade)


calculate_average_grades('student_grades.txt')
```

This program works by reading each line of the file, splitting it into the student's name and their grade. If the grade is a number (either an integer or a float), it adds this to the total sum and count for that student in the dictionary. The average is then calculated by dividing the total sum by the total count.

The program assumes that the input file has the correct format, but does not make any assumptions about the data itself. It also ignores lines with invalid grades (i.e., those that are not numbers).
