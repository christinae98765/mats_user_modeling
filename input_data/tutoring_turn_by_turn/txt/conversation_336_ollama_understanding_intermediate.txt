### Assistant: Write a recursive function to flatten a nested list in Python, where each sublist can be arbitrarily deep and contain any type of element (strings, integers, lists, etc.). The function should return a new list with all elements from the original nested list. 

Here's an example of what you might get as input: `[1, 2, [3, 4], ['a', 'b', [5, 6]], 7]`

### Human: def flatten(nested_list):
    result = []
    for element in nested_list:
        if isinstance(element, list):
            result.extend(flatten(element))
        else:
            result.append(element)
    return result

print(flatten([1, 2, [3, 4], ['a', 'b', [5, 6]], 7])) # Output: [1, 2, 3, 4, 'a', 'b', 5, 6, 7]

### Assistant: Your function seems to work as expected! It correctly flattens the input list and returns a new list with all elements.

For a medium-level question:

Write a function that takes a string as input and returns its reverse. However, there's an additional requirement: if the string contains any vowels (a, e, i, o, u), you must remove them from the reversed string before returning it.

Assume the input string only contains alphabetic characters and spaces.

### Human: Here is a Python function that fulfills your requirements:

```python
def reverse_and_remove_vowels(s):
    vowels = 'aeiouAEIOU'
    reversed_s = s[::-1]
    
    # Remove vowels from the reversed string
    for char in list(reversed_s):
        if char not in vowels:
            reversed_s = reversed_s.replace(char, '', 1)
    
    return ''.join(reversed_s.split())
```

This function works by first reversing the input string. It then iterates over each character in the reversed string and removes it if it's a vowel. Finally, it joins any remaining characters back together with an empty string to get the final result.

Write a function that takes a list of integers as input and returns the indices of all occurrences of the maximum element in the list.

### Human: def find_max_indices(nums):
    max_val = max(nums)
    return [i for i, num in enumerate(nums) if num == max_val]

Here's your next challenge:

Write a function that takes a list of strings as input and returns a new list containing only the strings that are palindromes (i.e., the same when reversed). The function should ignore case and consider only alphanumeric characters.

### Human: def filter_palindromes(strings):
    def is_palindrome(s):
        s = ''.join(c for c in s if c.isalnum()).lower()
        return s == s[::-1]
    
    return [s for s in strings if is_palindrome(s)]
