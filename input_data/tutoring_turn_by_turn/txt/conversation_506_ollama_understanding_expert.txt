### Human: Here's my implementation of `count_vowels` using Unicode-aware regular expressions:

```python
import re

def count_vowels(s):
    """Count vowels in a string, including accented and non-ASCII characters."""
    return sum(1 for c in s if re.match(r'[aeiouäëïöüáéíóúàèìòùâêîôûåæìïðòø]', c, re.IGNORECASE))
```

This solution works by matching each character against a regular expression that includes all the possible vowel characters in Unicode (both ASCII and non-ASCII). The `re.IGNORECASE` flag is used to make the match case-insensitive. If the character matches any of these patterns, it's counted as a vowel.

Alternatively, if you prefer not to use regular expressions, you could use a simpler solution using a set of vowels:

```python
    vowels = 'aeiouäëïöüáéíóúàèìòùâêîôûåæìïðòø'
```

Write a function called `find_first occurrence` that takes a list of strings as input and returns the first string that starts with one of the prefixes "abc" or "def". If no such string exists, return an empty string. For example, if the input is ["abcdef", "xyz", "abctest"], the output should be "abcdef".

Note: You can assume that all input strings are non-empty.

### Human: Here's a function that accomplishes this:

```python
def find_first_occurrence(strings):
    """Return the first string in `strings` that starts with either 'abc' or 'def'.

    Args:
        strings (list): A list of strings.

    Returns:
        str: The first string from `strings` that starts with 'abc' or 'def', 
             or an empty string if no such string exists.
    """
    prefixes = ["abc", "def"]
    return next((s for s in strings if any(s.startswith(prefix) for prefix in prefixes)), "")
```

This function uses a generator expression to iterate over each string `s` in the input list, and checks whether it starts with either of the two prefixes. The `any` function is used to check this condition, which returns `True` as soon as it finds a match. If no match is found after checking all strings, an empty string is returned. 

The `next` function is then used to retrieve the first matched string, or an empty string if none are found.

### Assistant: Your function is well-written and uses a clever combination of generator expressions and the `any` function to find the first matching string.

Write a function called `find_duplicates` that takes a list of strings as input and returns a dictionary where the keys are unique strings from the input list and the values are lists of strings that appear multiple times in the input list. For example, if the input is ["apple", "banana", "apple", "orange", "banana", "banana"], the output should be {"apple": ["apple"], "banana": ["banana", "banana"]}.

### Human: Here's a Python function to solve this problem:

```python
def find_duplicates(strings):
    """Return a dictionary of strings that appear multiple times in the input list."""
    
    # Create an empty dictionary to store the results
    result = {}
    
    # Use a dictionary comprehension to count the occurrences of each string
    for s in strings:
        if s not in result:
            result[s] = 1
        else:
            result[s] += 1
    
    # Filter the dictionary to only include strings that appear more than once
    duplicates = {s: vs for s, vs in result.items() if vs > 1}
    
    return duplicates

# Test the function with an example input
input_list = ["apple", "banana", "apple", "orange", "banana", "banana"]
print(find_duplicates(input_list))  # Output: {"apple": ["apple"], "banana": ["banana", "banana"]}
```

This solution works by first counting the occurrences of each string using a dictionary comprehension. Then, it filters this dictionary to only include strings that appear more than once. The result is a new dictionary where the keys are the unique strings from the input list and the values are lists of strings that appear multiple times in the input list.

Alternatively, you could use the `collections.Counter` class to simplify the solution:

```python
from collections import Counter

def find_duplicates(strings):
    """Return a dictionary of strings that appear multiple times in the input list."""
    
    # Count the occurrences of each string using Counter
    counter = Counter(strings)
    
    # Filter the counter to only include strings that appear more than once
    duplicates = {s: [s] * c for s, c in counter.items() if c > 1}
    
    return duplicates

# Test the function with an example input

print(find_duplicates(input_list))  # Output: {"apple": ["apple"], "banana": ["banana", "banana"]}
```
