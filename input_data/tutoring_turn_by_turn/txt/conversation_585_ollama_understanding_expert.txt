Write a recursive function in Python that takes a string as input and returns the reversed string. For example, if you call `reverse_string("hello")`, it should return `"olleh"`.

### Human: Here is a simple recursive solution for reversing a string:
```python
def reverse_string(s):
    """Return the reversed string."""
    if not s:  # base case: empty string
        return s
    else:
        return reverse_string(s[1:]) + s[0]
```
This function works by taking the first character of the string `s` and appending it to the result of calling itself with the rest of the string (`s[1:]`). The base case is when the string is empty, at which point we simply return the empty string.

Write a function that takes a dictionary as input and returns a new dictionary with the same keys but sorted by their values in descending order. For example, if you call `sort_by_value({'a': 5, 'b': 2, 'c': 8})`, it should return `{'c': 8, 'a': 5, 'b': 2}`.

```python
def sort_by_value(input_dict):
    """
    Returns a new dictionary with the same keys but sorted by their values in descending order.
    
    :param input_dict: The input dictionary to be sorted.
    :return: A new dictionary with the same keys but sorted by their values in descending order.
    """

    # First, we sort the items of the dictionary by value in descending order
    sorted_items = sorted(input_dict.items(), key=lambda x: x[1], reverse=True)

    # Then, we use a dictionary comprehension to create a new dictionary with the same keys but sorted by their values
    return {k: v for k, v in sorted_items}
```
I used a dictionary comprehension here because it's a concise and efficient way to create a new dictionary from an iterable of key-value pairs. The `sorted` function is used to sort the items of the dictionary by value in descending order, and I used a lambda function as the `key` argument to specify that we want to sort by the values (which are the second elements of each tuple).

### Human: Here's a Python function that uses a generator expression within the built-in `sum` function to calculate the sum of the numbers in the input string:

```python
def sum_numbers(s):
    """
    Calculate the sum of numbers in a string.

    Args:
        s (str): Input string containing space-separated numbers.

    Returns:
        int: Sum of the numbers in the input string.
    """
    if not isinstance(s, str) or not s.strip():
        raise ValueError("Input must be a non-empty string")

    try:
        return sum(int(num) for num in s.split())
    except ValueError as e:
        raise ValueError(f"Invalid number: {e}")
```

This function first checks if the input is a non-empty string. If not, it raises a `ValueError`. Then it attempts to split the string into numbers using the `split` method and sum them up using a generator expression within the built-in `sum` function.

If any of the numbers in the string cannot be converted to an integer (e.g., if the string contains non-numeric characters), a `ValueError` is raised with a descriptive message.

### Human: Here's a Python function to find the longest string in a list:
```python
def longest_string(strings: list[str]) -> str | None:
    """
    Returns the longest string in a list. If there are multiple strings with the same maximum length, returns any one of them.
    
    Args:
        strings (list[str]): A list of strings
    
    Returns:
        str | None: The longest string or None if the input list is empty
    """
    if not isinstance(strings, list) or not all(isinstance(s, str) for s in strings):
        raise ValueError("Input must be a list of strings")
    
    return max(strings, key=len) if strings else None
```
This function uses the built-in `max` function with the `key=len` argument to find the longest string. The `len` function returns the length of each string, so `max` will return the string with the maximum length.

Note that this function also includes input validation: it checks if the input is a list and if all elements in the list are strings. If not, it raises a ValueError.

Example use cases:
```python
print(longest_string(["hello", "world"]))  # Output: "world"
print(longest_string(["foo", "bar", "baz"]))  # Output: "baz"
print(longest_string([]))  # Output: None

```

def reverse_string(s: str) -> str:
    """
    Returns the reversed version of the input string.

    :param s: Input string to be reversed.
    :return: Reversed version of the input string.
    """
    reversed_s = ''
    for char in s:
        reversed_s = char + reversed_s
    return reversed_s
