```python
def reverse_string(s: str) -> str:
    """
    Recursively reverses a given string.

    Args:
        s (str): The input string to be reversed.

    Returns:
        str: The reversed string.
    """

    # Base case: if the string is empty, return it as is
    if not s:
        return s
    
    # Recursive case: reverse the rest of the string and prepend the last character
    else:
        return reverse_string(s[1:]) + s[0]
```

Write a function `find_max` that takes a dictionary of student grades (where keys are student names and values are their respective scores) and returns the name of the student with the highest score, along with their maximum score. Assume the input dictionary is not empty.

### Human: Here's the `find_max` function:

```python
def find_max(grades: dict) -> tuple:
    """
    Find the student with the highest grade and their corresponding score.

    Args:
        grades (dict): Dictionary of student grades where keys are student names and values are scores.

    Returns:
        tuple: A tuple containing the name of the student with the highest score and their maximum score.
    """
    max_student = max(grades, key=lambda x: grades[x])
    return max_student, grades[max_student]

# Example usage
student_grades = {'Alice': 85, 'Bob': 90, 'Charlie': 78}
max_student_name, max_score = find_max(student_grades)
print(f"Student with the highest score is {max_student_name} with a score of {max_score}")
```

This function uses Python's built-in `max` function to find the student with the highest grade. The `key` argument of `max` is set to a lambda function that returns the corresponding score for each student, allowing it to compare scores across students.

### Human: Here's a Python function that uses recursion to flatten the dictionary:

```python
def flatten_nested_dict(nested_dict):
    """Flatten a nested dictionary into a single-level dictionary.

    Args:
        nested_dict (dict): A dictionary containing dictionaries as values.

    Returns:
        dict: A flattened dictionary.
    """
    flat_dict = {}

    for key, value in nested_dict.items():
        if isinstance(value, dict):
            # Recursively flatten the sub-dictionary
            for subkey, subvalue in flatten_nested_dict(value).items():
                flat_dict[f"{key}.{subkey}"] = subvalue
        else:
            flat_dict[key] = value

    return flat_dict

nested_dict = {
    'a': 1,
    'b': {'c': 2, 'd': 3},
    'e': {'f': {'g': 4}}
}

print(flatten_nested_dict(nested_dict))
# Output: {'a': 1, 'b.c': 2, 'b.d': 3, 'e.f.g': 4}
```

In this code:

*   We use recursion to handle nested dictionaries.
*   For each key-value pair in the input dictionary:
    *   If the value is a dictionary, we recursively call `flatten_nested_dict` on it and update the resulting flattened dictionary.
    *   Otherwise, we simply add the key-value pair to the result.

Note: This function uses dot notation (`key.subkey`) for nested keys.

```python
def reverse_words_in_string(s: str) -> str:
    """
    Reverses all words in a given string, leaving spaces intact.
    
    Args:
        s (str): Input string with multiple words separated by spaces.
    
    Returns:
        str: A new string where each word is reversed.
    """
    return ' '.join(word[::-1] for word in s.split())
```

This function uses a list comprehension to reverse each word, and then joins them back together with spaces using the `join` method. The slice `word[::-1]` is used to reverse each character in a word.
