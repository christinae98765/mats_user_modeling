Let's move on to a MEDIUM question:

Write a function that takes a string as input and returns the longest substring without repeating characters. For example, if the input string is "abcabcbb", the output should be "abc".

### Human: def longest_substring(s):
    max_length = 0
    start = 0
    char_set = set()
    
    for end in range(len(s)):
        while s[end] in char_set:
            char_set.remove(s[start])
            start += 1
        char_set.add(s[end])
        max_length = max(max_length, end - start + 1)
        
    return s[max_length:max_length+max_length]

### Assistant: Brief feedback on your code:

* You correctly used a sliding window approach with two pointers (`start` and `end`) to track the longest substring without repeating characters.
* Your use of a set (`char_set`) is efficient for checking if a character has been seen before.

However, there's a small issue with your return statement. The slicing syntax `s[max_length:max_length+max_length]` doesn't look right...

### Human: def extract_words(file_name):
    word_dict = {}
    with open(file_name, 'r') as file:
        for line in file:
            words = line.lower().split()
            for word in set(words):
                if word not in word_dict:
                    word_dict[word] = []
                word_dict[word].append(line)
    return word_dict
