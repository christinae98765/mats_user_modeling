    # Split into words

    # Define stop words

            if word in word_counts:
                word_counts[word] += 1
            else:
                word_counts[word] = 1
    
    # Find the most common word

    most_common_word = [word for word, count in word_counts.items() if count == max_count]

Write a function called `count_unique_chars` that takes a string as input and returns the number of unique characters in the string. You can use any string methods you like, but don't use lists or sets.

### Human: ```python
def count_unique_chars(input_string):
    # Create an empty dictionary to store character counts
    char_counts = {}
    
    # Iterate over each character in the input string
    for char in input_string:
        # If the character is already in the dictionary, increment its count
        if char in char_counts:
            char_counts[char] += 1
        # Otherwise, add it to the dictionary with a count of 1
        else:
            char_counts[char] = 1
    
    # The number of unique characters is the total number of characters minus the sum of their counts
    return len(input_string) - sum(char_counts.values())

# Test the function

```

### Assistant: Your code uses a dictionary to count character occurrences, but it's not necessary to subtract the sum of counts from the total length; you can simply return the number of keys in the dictionary.
