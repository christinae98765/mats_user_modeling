def remove_vowels(s: str) -> str:
    """Remove all vowels from the input string."""

    return ''.join([c for c in s if c not in vowels])

### Human: Here's a solution for the `parse_data` function:

```python
import csv

def parse_data(filename, column_names):
    """
    Parse CSV data into a dictionary.

    Args:
        filename (str): Path to the CSV file.
        column_names (list[str]): List of column names to extract.

    Returns:
        dict: Dictionary where each key is a column name and each value is a list of corresponding values.
    """

    # Initialize an empty dictionary with the given column names
    data = {name: [] for name in column_names}

    # Use the `csv.reader` class to read the CSV file, skipping any header rows or comments
    with open(filename, 'r') as f:
        reader = csv.reader(f)
        next(reader)  # Skip the first row (header)

        for row in reader:
            try:
                # Extract corresponding values from each column and add them to the dictionary
                for name, value in zip(column_names, row):
                    data[name].append(value)
            except ValueError as e:
                print(f"Error processing row: {e}")

    return data
```

This function uses a `csv.reader` object to read the CSV file, skipping any header rows or comments. It then iterates over each row in the CSV file and extracts corresponding values from each column using `zip`. If there's an error extracting values (e.g., mismatched number of columns), it catches the exception and prints a warning message.

You can call this function with the example arguments:

```python
print(parse_data('data.csv', ['name', 'age']))
```

This should output something like:

```python

    'name': ['John', 'Jane', 'Bob'],
    'age': [25, 30, 35]
}
```
