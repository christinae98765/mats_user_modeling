Write a Python script that reads a text file containing a list of phone numbers, one number per line. It should then create a dictionary where each key is a phone number from the file, and its corresponding value is the count of occurrences of that number in the file.

Example input:
```python
123-4567
987-6543
123-4567
987-6543
765-4321
```
Assume the text file is named `phone_numbers.txt` and it's located in the same directory as your script.

### Human: Here's a Python script that accomplishes the task:

```python
import os

def count_phone_numbers(file_name: str) -> dict:
    """
    Reads phone numbers from a file, counts their occurrences, and returns them as a dictionary.

    Args:
        file_name (str): Name of the text file containing phone numbers.

    Returns:
        dict: Dictionary where each key is a phone number and its corresponding value is the count.
    """

    try:
        with open(file_name, 'r') as file:
            lines = [line.strip() for line in file.readlines()]
            return {phone_number: lines.count(phone_number) for phone_number in set(lines)}
    except FileNotFoundError:
        print(f"File '{file_name}' not found.")
        return {}

def main():
    """
    Entry point of the program. Reads phone numbers from `phone_numbers.txt`, counts their occurrences,
    and prints the result.
    """

    file_name = 'phone_numbers.txt'
    if os.path.exists(file_name):
        phone_counts = count_phone_numbers(file_name)
        for phone_number, count in phone_counts.items():
            print(f"{phone_number}: {count}")
    else:
        print(f"File '{file_name}' not found.")

if __name__ == "__main__":
    main()
```

This script uses a `count_phone_numbers` function to read the phone numbers from the file and create a dictionary with their counts. It also includes error handling for cases where the file is not found.

The `main()` function serves as the entry point of the program, which checks if the file exists before reading its contents. If the file is found, it calls `count_phone_numbers` to get the phone number counts and prints them out.

* Your use of a separate function `count_phone_numbers()` to handle the logic is great.
* The docstring for this function provides clear information about its purpose and parameters.
* You've also included error handling for cases where the file is not found.

Write a function that takes a string as input and returns the string with all vowels removed. For example, if the input is "hello world", the output should be "hll wrld".

### Human: Here's the Python code to solve this problem:
```python
def remove_vowels(s: str) -> str:
    """
    Returns a string with all vowels removed.
    
    :param s: Input string
    :return: String with vowels removed
    """
    vowels = 'aeiouAEIOU'
    return ''.join([c for c in s if c not in vowels])
```
Explanation:

This function uses a list comprehension to iterate over each character `c` in the input string `s`. If `c` is not a vowel (i.e., it's not in the string of vowels), it gets included in the resulting list. Finally, we use `''.join(...)` to concatenate these characters back into a single string.

You can test this function with the example input: `print(remove_vowels("hello world"))`

Write a Python program that asks the user for their age, then prints out a message based on whether they're an adult (18 or older) or a minor. If the user enters something other than a number, ask them to try again until they enter a valid age.

### Human: Here's a well-structured and readable program that meets your requirements:

```python
def get_age():
    """
    Asks the user for their age and returns it as an integer.

    If the input is not a valid number, keeps asking until a valid age is provided.
    """
    while True:
        try:
            age = int(input("Enter your age: "))
            if age < 0:
                print("Age cannot be negative. Please try again.")
            else:
                break
        except ValueError:
            print("That's not a valid number. Try again!")

    return age


def is_minor(age):
    """
    Returns True if the given age indicates that the person is a minor (under 18).
    """
    return age < 18


def main():
    age = get_age()
    if is_minor(age):
        print(f"You are a minor. You must be at least 18 to be an adult.")
    else:
        print(f"Welcome! You're an adult.")


if __name__ == "__main__":
    main()
```

In this code, I used a `try`/`except` block in the `get_age()` function to catch any `ValueError` exceptions that occur when the user enters something other than a number. This allows me to keep asking for input until a valid age is provided.

I also defined a separate `is_minor()` function to make the code more modular and easier to understand. This function takes an age as input and returns a boolean indicating whether it's a minor or not.

Finally, in the `main()` function, I call `get_age()` to get the user's age and then print out a message based on their status (minor or adult).

Here's some brief feedback:

* Your use of `try`/`except` blocks to handle invalid input is excellent.

Write a Python program that takes a string as input, then prints out each word in the string, followed by its length. For example, if the input is "hello world", the output should be:
```
hello 5
world 5
```

### Human: Here's a Python program that accomplishes this:

```python
def process_string(input_str):
    words = input_str.split()
    for word in words:
        print(f"{word} {len(word)}")

input_str = input("Enter a string: ")
process_string(input_str)
```

This code works as follows:

* The `split()` method splits the input string into a list of words.
* It then iterates over each word and prints it, followed by its length using an f-string.

Note that this code uses the f-string feature to create formatted strings. This is a more readable way to insert variables into strings compared to concatenation or string formatting methods like `str.format()`.

Here's some brief feedback:

* Your use of `split()` to split the input string into words is concise.
