### Human: Here's a concise and efficient solution using Python:

```python
def count_vowels(s):
    vowels = set('aeiouAEIOU')
    return {
        'vowels': sum(1 for c in s if c in vowels),
        'consonants': len(s) - sum(1 for c in s if c in vowels)
    }
```

This function uses a generator expression to iterate over the input string `s` and count the occurrences of vowels using the `sum()` function. The total count of consonants is then obtained by subtracting the vowel count from the length of the input string.

```python
def count_vowels(s):

        'vowels': len([c for c in s if c in vowels]),
```

Write a function `insertion_sort(arr)` that takes a list of integers as input and returns a new sorted list in ascending order using the insertion sort algorithm. The function should not modify the original list.

(Note: I'm expecting a solution with a clear explanation of how the insertion sort algorithm works, including examples or comments to illustrate its steps.)

### Human: Here's a Python implementation of the `insertion_sort` function:

```python
def insertion_sort(arr):
    """
    Returns a new sorted list in ascending order using the insertion sort algorithm.

    The insertion sort algorithm works by iterating over the input list and inserting each element into its correct position within the already-sorted portion of the list.
    This is achieved by comparing each element to all preceding elements, shifting larger elements one position to the right until a smaller element is found.

    :param arr: A list of integers
    :return: A new sorted list in ascending order
    """
    # Create a copy of the original list to avoid modifying it
    sorted_arr = arr.copy()

    for i in range(1, len(sorted_arr)):
        key = sorted_arr[i]  # Store current element as 'key'
        
        # Compare the key with all preceding elements and shift larger elements one position to the right
        j = i - 1
        while j >= 0 and sorted_arr[j] > key:
            sorted_arr[j + 1] = sorted_arr[j]
            j -= 1
        
        # Insert the key at its correct position within the already-sorted portion of the list
        sorted_arr[j + 1] = key

    return sorted_arr


# Example usage:
numbers = [64, 34, 25, 12, 22, 11, 90]
print("Original List:", numbers)
sorted_numbers = insertion_sort(numbers)
print("Sorted List:", sorted_numbers)
```

This implementation follows the standard steps of the insertion sort algorithm:

1.  Iterate over the input list starting from the second element (index 1).
2.  Store each current element as a 'key' to be inserted into its correct position.
3.  Compare the key with all preceding elements and shift larger elements one position to the right until a smaller element is found or the beginning of the list is reached.
4.  Insert the key at its correct position within the already-sorted portion of the list.

The function returns a new sorted list in ascending order without modifying the original input list. The example usage demonstrates how to use this function with a sample list of integers.

(Note: I'm expecting a solution with a clear explanation of how to use loops or other methods to iterate over the characters in the string.)

### Human: Here's a function that accomplishes this task using Python:
```python
def count_occurrences(s, char):
    """
    Count occurrences of a character within a given string.

    Args:
        s (str): Input string.
        char (str): Character to search for.

    Returns:
        int: Number of times the specified character appears in the input string.
    """
    if not isinstance(s, str) or not isinstance(char, str):
        raise ValueError("Inputs must be strings")

    return sum(1 for c in s.lower() if c == char.lower())
```
Here's how it works:

1. First, we check that both inputs are indeed strings. If not, a `ValueError` is raised.
2. We then use a generator expression to iterate over each character `c` in the input string `s`, converted to lowercase using the `lower()` method.
3. For each character, we check if it matches the target character `char`, also converted to lowercase for case-insensitive matching.
4. The `sum()` function counts the number of characters that match the target character.

Note: This solution uses a generator expression instead of a loop because it's more concise and efficient in Python.
