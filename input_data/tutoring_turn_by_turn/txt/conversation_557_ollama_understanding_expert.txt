```python
def get_even_numbers(numbers):
    """
    Returns a new list containing only the even numbers from the input list.
    
    Args:
        numbers (list): A list of integers.
    
    Returns:
        list: A list of even integers.
    """

    # Initialize an empty list to store the even numbers
    even_nums = []

    # Iterate over each number in the input list
    for num in numbers:
        # Check if the current number is even by using the modulus operator (%)
        # If a number is even, its remainder when divided by 2 will be 0
        if num % 2 == 0:
            # Append the even number to the result list
            even_nums.append(num)

    # Return the list of even numbers
    return even_nums

# Example usage:
numbers = [1, 2, 3, 4, 5, 6]
print(get_even_numbers(numbers))  # Output: [2, 4, 6]
```

However, it's worth noting that this can be achieved more concisely using a list comprehension:

```python
def get_even_numbers(numbers):
    return [num for num in numbers if num % 2 == 0]

# Example usage:

print(get_even_numbers(numbers))  # Output: [2, 4, 6]
```

```python
import re
from collections import Counter

def find_duplicates(text):
    # Remove punctuation and convert to lowercase
    words = re.sub(r'[^\w\s]', '', text).lower().split()

    # Use Counter to count word occurrences
    word_counts = Counter(words)

    duplicates = [(word, count) for word, count in word_counts.items() if count > 1]
```

```python
def find_duplicates(text):
```

```python
    Args:
        list1 (list): The first sorted list.
        list2 (list): The second sorted list.

    Returns:
        list: A new sorted list containing all elements from both input lists.

    # Initialize variables to keep track of the current position in each list

            merged_list.append(list1[i])
```

Write a function that takes a string as input and returns a new string with all vowels removed. Assume you have a set of standard English vowels (a, e, i, o, u). Use any Python data structure or method to solve this problem.

### Human: Here's the solution:
```python
def remove_vowels(s: str) -> str:
    """
    Removes all vowels from the input string.
    
    Args:
        s (str): The input string.
    
    Returns:
        str: A new string with all vowels removed.
    """
    vowels = set('aeiou')
    return ''.join(c for c in s if c.lower() not in vowels)
```
This function uses a generator expression to iterate over each character in the input string, and an f-string is used to join the characters together into a new string. The `lower()` method is used to ensure that both lowercase and uppercase vowels are removed.

Alternatively, you can use the `filter()` function or list comprehension with a conditional statement for a more explicit solution:


```python
def remove_char(input_str: str, char: str) -> str:
    """
    Returns a new string with all occurrences of 'char' removed from 'input_str'.

    Args:
        input_str (str): The original string.
        char (str): The character to be removed.

    Returns:
        str: A new string without the specified character.
```
