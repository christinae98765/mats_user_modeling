Write a recursive function that flattens a nested list structure into a single list. For example, if the input is `[1, [2, 3], [4, [5, 6]]]`, the output should be `[1, 2, 3, 4, 5, 6]`. The function should work with any level of nesting.

### Human: Here's a recursive solution to flatten the nested list:

```python
def flatten(nested_list):
    flat_list = []
    
    for element in nested_list:
        if isinstance(element, list):
            flat_list.extend(flatten(element))
        else:
            flat_list.append(element)
            
    return flat_list

nested_input = [1, [2, 3], [4, [5, 6]]]
print(flatten(nested_input))  # Output: [1, 2, 3, 4, 5, 6]
```

Implement a function that takes two dictionaries as input and returns the intersection of their keys, i.e., the common keys between both dictionaries. The function should handle cases where one dictionary has fewer keys than the other and should return an empty list if there are no common keys. You can assume all key-value pairs in the dictionaries have unique keys.

(And don't worry, I'm not going to ask for an easier or harder question based on your previous requests!)

### Human: Here's a function that accomplishes this task:

```python
def intersection_of_keys(dict1, dict2):
    return [key for key in dict1.keys() if key in dict2]
```

This function works by using list comprehension to iterate over the keys of the first dictionary. It checks each key to see if it exists in the second dictionary's keys and includes it in the new list if so. This way, we get a list of common keys between the two dictionaries.

However, this solution has an issue with performance because it potentially iterates over all keys in both dictionaries for every unique key found in the first dictionary. A more efficient approach would be to convert one of the dictionaries' keys to a set and then use that to filter out common keys from the other dictionary's keys. Here is how you could implement this:

```python
def intersection_of_keys(dict1, dict2):
    return [key for key in dict1.keys() if key in set(dict2.keys())]
```

But there is another way to do it using set intersection operation directly on dictionaries' keys. This approach will work well when dealing with large datasets as it has a better time complexity.

```python
def intersection_of_keys(dict1, dict2):
    return list(set(dict1.keys()).intersection(set(dict2.keys())))
```

Here it is:

Implement a function that reads the contents of a file (specified by its path) and returns a dictionary where each key-value pair represents a word and its frequency in the document. The function should be case-insensitive and should handle punctuation marks correctly.

### Human: def read_file_frequency(file_path):
    try:
        with open(file_path, 'r') as file:
            contents = file.read()
    except FileNotFoundError:
        print(f"File {file_path} not found.")
        return {}
    
    # Convert to lower case and remove punctuation
    contents = ''.join(e for e in contents if e.isalnum() or e.isspace()).lower()
    
    # Split into words
    words = contents.split()
    
    # Initialize frequency dictionary
    frequency_dict = {}
    
    for word in words:
        # Increment frequency of each word
        if word in frequency_dict:
            frequency_dict[word] += 1
        else:
            frequency_dict[word] = 1
    
    return frequency_dict
