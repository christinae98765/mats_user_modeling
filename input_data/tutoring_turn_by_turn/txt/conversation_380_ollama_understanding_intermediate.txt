### Human: ```python
def get_older_than_30(dict1, dict2):
    """
    Returns a list of all people older than 30 along with their ages.
    
    Args:
        dict1 (dict): First dictionary containing person's name and age.
        dict2 (dict): Second dictionary containing person's name and age.
        
    Returns:
        list: List of tuples where each tuple contains the name and age of a person older than 30.
        
    Raises:
        ValueError: If any dictionary contains "age" with non-integer value.
    """

    # Check if both dictionaries contain 'name' and 'age'
    if not ('name' in dict1 and 'age' in dict1) or not ('name' in dict2 and 'age' in dict2):
        raise ValueError("Both dictionaries must contain 'name' and 'age'")

    # Initialize an empty list to store the result
    older_than_30 = []

    # Check if 'age' values are integers for both dictionaries
    if not isinstance(dict1['age'], int) or not isinstance(dict2['age'], int):
        raise ValueError("Age value must be an integer")

    # Iterate over both dictionaries
    for person in [dict1, dict2]:
        
        # Check if the age is greater than 30
        if person['age'] > 30:
            
            # If it is, append a tuple containing name and age to the result list
            older_than_30.append((person['name'], person['age']))

    # Return the result list
    return older_than_30
```

Implement a recursive function called `flatten_list` that takes a nested list as input and returns a new flat list containing all the elements from the original list, without any nesting. For example, if the input is `[1, [2, 3], [4, [5]]]`, the output should be `[1, 2, 3, 4, 5]`. The function should handle arbitrarily nested lists of arbitrary depth.

### Human: def flatten_list(nested_list):
    flat_list = []
    for element in nested_list:
        if isinstance(element, list):
            flat_list.extend(flatten_list(element))
        else:
            flat_list.append(element)
    return flat_list

# Example usage
print(flatten_list([1, [2, 3], [4, [5]]])) # Output: [1, 2, 3, 4, 5]

### Assistant: Your `flatten_list` function is a good attempt at solving the problem. It correctly checks if each element is a list and recursively calls itself to flatten any nested lists.
