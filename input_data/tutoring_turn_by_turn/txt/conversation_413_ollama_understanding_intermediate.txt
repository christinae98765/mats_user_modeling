Write a function that takes a string as input and returns a new string with all consecutive duplicate characters removed. For example, if the input is "hellooo world", the output should be "helo worl".

### Human: ```python
def remove_consecutive_duplicates(input_str):
    result = []
    for char in input_str:
        if not result or char != result[-1]:
            result.append(char)
    return ''.join(result)

# Test the function
print(remove_consecutive_duplicates("hellooo world"))  # Outputs: "helo worl"
```
