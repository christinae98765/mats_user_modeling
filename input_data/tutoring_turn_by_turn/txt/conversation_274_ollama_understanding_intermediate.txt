Write a function that takes a dictionary of students' scores, where each key is a student's name and each value is another dictionary containing their scores for three subjects (e.g., "Math", "Science", "English"). The function should return the average score of all students, as well as a list of students who scored above the class average.

For example, if you have:

`students = {"Alice": {"Math": 85, "Science": 90, "English": 78}, 
             "Bob": {"Math": 92, "Science": 88, "English": 95},
             "Charlie": {"Math": 76, "Science": 80, "English": 90}}`

Your function should return:

`{"average_score": X, "top_students": ["Bob"]}`

where `X` is the average score of all students.

### Human: def calculate_average_and_top_students(students):
    total_scores = {}
    for student in students:
        scores = 0
        for subject in students[student]:
            scores += students[student][subject]
        total_scores[student] = scores / len(students[student])
    
    average_score = sum(total_scores.values()) / len(total_scores)
    top_students = [student for student, score in total_scores.items() if score > average_score]
    
    return {"average_score": average_score, "top_students": top_students}

students = {"Alice": {"Math": 85, "Science": 90, "English": 78}, 
            "Bob": {"Math": 92, "Science": 88, "English": 95},
            "Charlie": {"Math": 76, "Science": 80, "English": 90}}

print(calculate_average_and_top_students(students))

Now, let's move on to a MEDIUM question:

Write a function that takes a string `s` as input and returns a new string where all words are reversed. For example, if `s = "Hello world"`, the output should be `"olleH dlrow"`.

### Human: Here is a Python solution for reversing each word in the given string:
```python
def reverse_words(s):
    # Split the input string into a list of words
    words = s.split()

    # Use a list comprehension to reverse each word and join them back together
    reversed_s = ' '.join([word[::-1] for word in words])

    return reversed_s

# Example usage:
s = "Hello world"
print(reverse_words(s))  # Output: "olleH dlrow"
```

Now, let's move on to a MEDIUM question:

Given a list of numbers `numbers = [1, 2, 3, 4, 5]`, write a function that returns the sum of all even numbers in the list.

### Human: Here is a Python function to calculate the sum of all even numbers in the given list:
```python
def sum_even_numbers(numbers):
    return sum(num for num in numbers if num % 2 == 0)
```
This function uses a generator expression within the built-in `sum()` function to iterate over each number in the input list and add up only those that are even (i.e., where `num % 2 == 0`).

### Assistant: Your code is concise and effectively uses a generator expression to sum up even numbers in the input list.
