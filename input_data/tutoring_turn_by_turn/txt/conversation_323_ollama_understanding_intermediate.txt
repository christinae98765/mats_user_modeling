Here's a challenging question:

Implement a function that takes a string as input and returns a dictionary where each key is a word from the input string and its corresponding value is the frequency of that word. The function should be case-insensitive and ignore punctuation.

### Human: def word_frequency(input_string):
    # Convert to lower case
    input_string = input_string.lower()
    
    # Remove punctuation
    import string
    translator = str.maketrans('', '', string.punctuation)
    input_string = input_string.translate(translator)
    
    # Split into words
    words = input_string.split()
    
    # Create dictionary to store word frequencies
    frequency_dict = {}
    
    for word in words:
        if word not in frequency_dict:
            frequency_dict[word] = 1
        else:
            frequency_dict[word] += 1
    
    return frequency_dict

### Assistant: Your code is good, but you might want to consider using a `defaultdict` from the `collections` module to simplify the word frequency counting.
