Here's a HARD question:

Write a function called `find_max_depth` that takes an arbitrarily nested list of integers as input and returns the maximum depth of nesting. For example, if the input is `[1, [2, [3, 4]], [5]]`, the output should be `3`. If the input is `[1]`, the output should be `1`.

Can you give it a try?

### Human: def find_max_depth(nested_list):
    def helper(sublist, depth=1):
        max_depth = depth
        for item in sublist:
            if isinstance(item, list):
                max_depth = max(max_depth, helper(item, depth + 1))
        return max_depth
    return helper(nested_list)

### Assistant: Your function looks good! You're using a helper function to recursively traverse the nested list and keep track of the maximum depth.

### Human: Here's a function that uses a while loop to reverse a string:

```python
def reverse_string(s):
    reversed_s = ""
    i = len(s) - 1
    
    while i >= 0:
        reversed_s += s[i]
        i -= 1
        
    return reversed_s

# Test the function

```

Here's a MEDIUM question:

Write a function called `most_frequent` that takes a list of strings as input and returns the most frequently occurring string in the list. For example, if the input is `["apple", "banana", "apple", "orange"]`, the output should be `"apple"`. You can use Python's built-in functions to solve this problem.

### Human: def most_frequent(strings):
    freq_dict = {}
    
    for string in strings:
        if string in freq_dict:
            freq_dict[string] += 1
        else:
            freq_dict[string] = 1
    
    max_freq = max(freq_dict.values())
    most_common = [string for string, freq in freq_dict.items() if freq == max_freq]
    
    return most_common[0]

print(most_frequent(["apple", "banana", "apple", "orange"]))  # Output: apple
