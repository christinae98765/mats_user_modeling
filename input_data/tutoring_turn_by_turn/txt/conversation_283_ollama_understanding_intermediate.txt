Here's a slightly more challenging question:

Write a function that takes two lists as input and returns a new list containing only the elements common to both input lists, without duplicates. 

(Note: I'm aware you might want something easier or harder, but this one should be just right!)

### Human: Here is the solution:
```python
def common_elements(list1, list2):
    return list(set(list1) & set(list2))
```
This function uses Python's built-in `set` data structure to find the intersection of the two input lists. The `&` operator performs a set intersection, and then we convert it back to a list using the `list()` constructor.

I've tested this with some example inputs:
```python
print(common_elements([1, 2, 3], [2, 3, 4]))  # Output: [2, 3]
print(common_elements(['a', 'b', 'c'], ['b', 'c', 'd']))  # Output: ['b', 'c']
```

Write a function that takes a list of numbers as input and returns the sum of all even numbers in the list.

### Human: def sum_even_numbers(numbers):
    total = 0
    for num in numbers:
        if num % 2 == 0:
            total += num
    return total

Write a function that takes a string as input and returns a new string with all occurrences of a given character removed. The function should be case-insensitive and ignore non-alphanumeric characters.

### Human: def remove_char(input_string, char):
    result = ""
    for c in input_string:
        if not c.isalnum() or c.lower() != char.lower():
            result += c
    return result

Write a function that takes a dictionary as input where each key is a person's name and each value is another dictionary representing their address (with keys 'street', 'city', and 'state'). The function should return a new dictionary with the same people, but with their addresses sorted by city.

(Note: I know you asked for something different, but this one's a good challenge!)

### Human: def sort_by_city(ppl_dict):
    sorted_ppl = {}
    
    # Sort the list of cities
    cities = []
    for person in ppl_dict:
        city = ppl_dict[person]['city']
        if city not in cities:
            cities.append(city)
            
    cities.sort()
    
    # For each city, make a new list of people with that city
    for city in cities:
        ppl_with_city = [ppl for ppl in ppl_dict if ppl_dict[ppl]['city'] == city]
        
        # Add these people to the sorted dictionary
        for person in ppl_with_city:
            sorted_ppl[person] = ppl_dict[person]
            
    return sorted_ppl
