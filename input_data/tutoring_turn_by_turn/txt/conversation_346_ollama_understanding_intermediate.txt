Write a program that can take a list of words as input and return a dictionary where each word is associated with its frequency in the list. For example, if you pass `['apple', 'banana', 'apple', 'orange']`, the output should be `{'apple': 2, 'banana': 1, 'orange': 1}`.

Note: You can assume that the input list will not contain any empty strings or duplicates of the same word.

### Human: def word_frequency(word_list):
    frequency_dict = {}
    
    for i in range(len(word_list)):
        if word_list[i] in frequency_dict:
            frequency_dict[word_list[i]] += 1
        else:
            frequency_dict[word_list[i]] = 1
    
    return frequency_dict

### Human: ```python
import re

def count_unique_words(filename):
    word_freq = {}
    with open(filename, 'r') as file:
        text = file.read().lower()
        words = re.findall(r'\b\w+\b', text)
        for word in words:
            if word not in word_freq:
                word_freq[word] = 1
            else:
                word_freq[word] += 1

    # Sort the dictionary by frequency and then by alphabetical order
    sorted_word_freq = dict(sorted(word_freq.items(), key=lambda item: (-item[1], item[0])))

    for word, freq in sorted_word_freq.items():
        print(f"({word}, {freq})")

count_unique_words('example.txt')
```

### Assistant: Your code looks good! You're correctly reading in the file, converting it to lowercase, extracting the words using regular expressions, and counting their frequencies.

I'd like to suggest a minor improvement: instead of loading the entire file into memory at once, you could process the file line by line using a `with open()` block. This would be more efficient for large files.

Now, let's move on to a MEDIUM question:

Write a function called `insert_element` that takes in a list and an element as input, inserts the element at a specified position (0-based index), and returns the modified list. For example, if you pass `[1, 2, 3]` and `4` with an insertion position of `1`, your output should be `[1, 4, 2, 3]`.

### Human: Here is a function that meets the requirements:
```python
def insert_element(lst, elem, pos):
    return lst[:pos+1] + [elem] + lst[pos+1:]
```
This function uses list slicing to split the input list into three parts: everything before the insertion position, the element itself, and everything after the insertion position. It then concatenates these three parts together in the correct order, effectively inserting the element at the specified position.

You can test this function with some examples:
```python
print(insert_element([1, 2, 3], 4, 1))  # Output: [1, 4, 2, 3]
print(insert_element(['a', 'b', 'c'], 'd', 0))  # Output: ['a', 'd', 'b', 'c']
```
