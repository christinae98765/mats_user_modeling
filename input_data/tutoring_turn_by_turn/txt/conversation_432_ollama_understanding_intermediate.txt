Write a function `count_unique_words` that takes a string of text as input and returns a dictionary where each key is a unique word from the text and each value is the frequency of that word. The function should be case-insensitive and ignore punctuation. What's your code?

### Human: def count_unique_words(text):
    words = {}
    
    for word in text.split():
        # Remove any leading or trailing punctuation
        stripped_word = ''.join(e for e in word if e.isalnum())
        
        # Convert to lower case to ensure the comparison is case-insensitive
        stripped_word = stripped_word.lower()
        
        # Count how many times this word has been encountered before
        words[stripped_word] += 1
    
    return words

Here's a HARD question:

Write a function `parse_csv_file` that takes a filename as input and returns a dictionary where each key is a column name and each value is a list of values from that column in the CSV file. The function should handle cases where there are missing or duplicate values in the file. What's your code?

### Human: def parse_csv_file(filename):
    data = {}
    with open(filename, 'r') as f:
        first_line = next(f).split(',')
        for i in range(len(first_line)):
            if not i in data:
                data[first_line[i]] = []
            for line in f:
                values = line.split(',')
                if i < len(values):
                    if len(data[first_line[i]]) == 0 or values[i] != data[first_line[i]][-1]:
                        data[first_line[i]].append(values[i])
    return data
