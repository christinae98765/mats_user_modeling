As per your request, I'll ask a HARD question:

Write a program that reads a text file and extracts all unique words from it, storing them in a dictionary where each word is a key and its frequency is the value. The program should ignore case and punctuation.

### Human: Here's the code for the task:
```python
import string

def extract_unique_words(filename):
    word_freq = {}
    with open(filename) as file:
        for line in file:
            words = line.lower().split()
            for word in words:
                word = word.strip(string.punctuation)
                if word not in word_freq:
                    word_freq[word] = 1
                else:
                    word_freq[word] += 1

    return word_freq

print(extract_unique_words('example.txt'))
```
This program reads a file line by line, splits each line into words (splitting on whitespace), removes punctuation from each word and counts the occurrences of unique words in a dictionary.

### Assistant: Your code is a good start, but it has some potential issues:

1. You're using `string.punctuation` to remove punctuation from each word, which will remove all punctuation at once. Instead, you might want to use a regular expression to match and replace individual punctuation characters.
2. Your function assumes that the input file exists in the same directory as your script. If it's not there, the program will crash.

Here's an MEDIUM question:

Write a function called `find_longest_word` that takes a list of strings (words) as input and returns the longest word(s). If there are multiple words with the same maximum length, return all of them.

### Human: def find_longest_word(words):
    max_length = 0
    longest_words = []
    
    for word in words:
        if len(word) > max_length:
            max_length = len(word)
            longest_words = [word]
        elif len(word) == max_length:
            longest_words.append(word)
    
    return longest_words
