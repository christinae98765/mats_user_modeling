```python
def remove_vowels(sentence: str) -> list[str]:
    """
    Returns a list of words with all vowels removed.

    Args:
        sentence (str): Input sentence.

    Returns:
        list[str]: List of words without vowels.
    """

    # Define the set of vowels for easy lookup
    vowels = {'a', 'e', 'i', 'o', 'u'}

    # Use a list comprehension to process each word in the sentence
    return ["".join(c for c in word if c.lower() not in vowels) 
            for word in sentence.split()]
```

This function splits the input `sentence` into words and uses another generator expression inside a list comprehension to create a new string with all vowels removed from each word.

```python
def merge_pairs(list1, list2):
    """
    Merge pairs of items from two lists into a single list.

    Args:
        list1 (list): List of pairs.
        list2 (list): List of pairs.

    Returns:
        list: New list with merged items.
```

Here's a MEDIUM question:

Write a function that takes a string `paragraph` as input and returns a list of sentences in the paragraph, where each sentence is split by punctuation marks (e.g., `.`, `,`, `?`, `!`). For example, if the input is "Hello, world! This is a test. How are you?", the output should be ["Hello, world!", "This is a test.", "How are you?"].

### Human: Here's a function that does exactly what you asked for:

```python
import re

def split_into_sentences(paragraph):
    """
    Splits a paragraph into sentences based on punctuation marks.
    
    Args:
        paragraph (str): The input string to be split.
    
    Returns:
        list: A list of sentences in the paragraph, each terminated by a punctuation mark.
    """
    # Regular expression pattern for one or more punctuation marks
    punct_pattern = r'[.!?]'
    
    # Split the paragraph into sentences using the regular expression
    sentences = re.split(punct_pattern, paragraph)
    
    # Filter out empty strings that might result from leading/trailing whitespace or consecutive punctuation marks
    return [sentence.strip() for sentence in sentences if sentence]

# Test the function with an example paragraph
paragraph = "Hello, world! This is a test. How are you?"
print(split_into_sentences(paragraph))
```

This will output: `['Hello, world!', 'This is a test.', 'How are you?']`. 

The regular expression `[.!?]` matches any of the punctuation marks `.`, `!`, or `?`. The `re.split()` function splits the paragraph into substrings at each occurrence of these punctuation marks.

```python
def repeat_numbers(numbers):
```

Now, let's move on to a MEDIUM question:

Write a function that takes a string `text` as input and returns the first occurrence of a word that has more than 3 characters. For example, if the input is "Hello world, this is a test", the output should be the index of the first such word (in this case, it would return 5, since "world" is the first word with more than 3 characters). If no such word exists, the function should return -1.

### Human: Here's my solution for this problem:

```python
def first_long_word(text):
    """
    Returns the index of the first word with more than 3 characters.
    
    Args:
        text (str): Input string
    
    Returns:
        int: Index of the first long word, or -1 if no such word exists
    """
    words = text.split()
    for i, word in enumerate(words):
        if len(word) > 3:
            return i
    return -1

# Alternative approach using a list comprehension and next function
def first_long_word_alt(text):
    """
    Returns the index of the first word with more than 3 characters.
    
    Args:
        text (str): Input string
    
    Returns:
        int: Index of the first long word, or -1 if no such word exists
    """
    words = [i for i, word in enumerate(text.split()) if len(word) > 3]
    return next((word for word in words if word is not None), -1)
```

In this solution, I've provided two approaches. The first one iterates over the list of words and returns as soon as it finds a word with more than 3 characters.

The second approach uses a list comprehension to create a list of indices where such words are found. If no such word exists, an empty list is returned. The `next` function then attempts to return the first element from this list (which will be `None` if the list was empty).
