Write a Python program that takes a string input from the user and returns the number of vowels in that string. For example, if the input is "hello world", the output should be 2.

### Human: def count_vowels(user_string):
    """
    This function counts the number of vowels in a given string.

    It works by iterating over each character in the string, checking if it's a vowel (i.e., 'a', 'e', 'i', 'o', or 'u'), and incrementing a counter for each one found.

    Parameters:
        user_string (str): The input string to count vowels from.

    Returns:
        int: The total number of vowels in the input string.
    """
    vowel_count = 0
    for char in user_string.lower():
        if char == 'a' or char == 'e' or char == 'i' or char == 'o' or char == 'u':
            vowel_count += 1
    return vowel_count

### Human: def capitalize_names(name_list):
    """
    This function takes a list of names as input and returns a new list 
    where each name is capitalized.

    Args:
        name_list (list): A list of names to be capitalized.

    Returns:
        list: A new list with all names capitalized.
    """
    
    # Initialize an empty list to store the capitalized names
    capitalized_names = []
    
    # Iterate over each name in the input list
    for name in name_list:
        # Capitalize the current name using the capitalize() method and add it to the list
        capitalized_names.append(name.capitalize())
    
    # Return the list of capitalized names
    return capitalized_names

Here's a HARD question:

Implement a function called `merge_json_files` that takes two JSON files as input and merges them into a single file. The merged file should have the same structure as the original files, but with all data from both files combined.

For example, if the input files are `file1.json`: `{ "name": "John", "age": 30 }` and `file2.json`: `{ "city": "New York", "country": "USA" }`, the output file should be `merged_file.json`: `{ "name": "John", "age": 30, "city": "New York", "country": "USA" }`.

Note that this function should handle nested structures (like objects inside objects) and arrays. Good luck!

### Human: Here's a solution to the problem:

```python
import json

def merge_json_files(file1, file2):
    with open(file1, 'r') as f1:
        data1 = json.load(f1)

    with open(file2, 'r') as f2:
        data2 = json.load(f2)

    def recursive_merge(data1, data2):
        for key in data2.keys():
            if key not in data1:
                data1[key] = data2[key]
            else:
                if isinstance(data1[key], dict) and isinstance(data2[key], dict):
                    recursive_merge(data1[key], data2[key])
                elif isinstance(data1[key], list) and isinstance(data2[key], list):
                    data1[key].extend(data2[key])
        return data1

    merged_data = recursive_merge(data1, data2)

    with open('merged_file.json', 'w') as f:
        json.dump(merged_data, f)
```

This function first loads the JSON files into Python dictionaries. It then defines a helper function `recursive_merge` that recursively merges two nested structures by adding new keys from the second dictionary and updating existing key values if necessary. Finally, it writes the merged data back to a new JSON file called 'merged_file.json'.
