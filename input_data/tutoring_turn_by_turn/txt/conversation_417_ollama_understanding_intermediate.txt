### Human: Here is the function that merges two dictionaries:
```python
def merge_dictionaries(dict1, dict2):
    new_dict = {}
    for key in set(list(dict1.keys()) + list(dict2.keys())):
        if key in dict1 and key in dict2:
            new_dict[key] = [dict1[key], dict2[key]]
        elif key in dict1:
            new_dict[key] = dict1[key]
        else:
            new_dict[key] = dict2[key]

    return new_dict
```
This function iterates over all unique keys from both input dictionaries, and for each key that exists in both dictionaries, it combines the corresponding values into a list. If a key only exists in one dictionary, its value is simply added to the output dictionary.
