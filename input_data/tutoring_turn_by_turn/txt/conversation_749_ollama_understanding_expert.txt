Here's a medium question:

Write a function `reverse_words()` that takes a string of words as input and returns a new string with all words reversed in order. For example, if the input is `"Hello World Python"`, the output should be `"olleH dlroW nohtyp"`.

### Human: Here's the solution:

```python
def reverse_words(s: str) -> str:
    """
    Reverses each word in a given string.

    Args:
        s (str): The input string of words.

    Returns:
        str: A new string with all words reversed.
    """
    if not isinstance(s, str):
        raise ValueError("Input must be a string")

    # Split the input into individual words
    words = s.split()

    # Use a list comprehension to reverse each word and join them back together
    reversed_s = ' '.join(word[::-1] for word in words)

    return reversed_s

# Example usage:
print(reverse_words("Hello World Python"))  # Output: "olleH dlroW nohtyp"
```

This function uses the `split()` method to split the input string into a list of individual words, and then employs a list comprehension with slicing (`[::-1]`) to reverse each word. Finally, it joins the reversed words back together using `' '.join()`. The result is returned as a new string.

Let's move on to a MEDIUM question:

Write a function that takes a string input and returns a new string with all vowels removed. For example, if the input is `"Hello World"`, the output should be `"Hll Wrld"`.

### Human: Here's the solution:

```python
def remove_vowels(s: str) -> str:
    """
    Returns a new string with all vowels removed from the input string.

    :param s: Input string
    :return: String without vowels
    """
    vowels = set("aeiouAEIOU")
    return "".join([c for c in s if c not in vowels])
```

This function uses a list comprehension to generate a new string where each character is included only if it's not a vowel. The `set` data structure is used for efficient lookups of vowels.

Implement a function that takes a string input and returns a dictionary where each key is a word from the input string, and each value is the number of times that word appears in the string. For example, if the input is `"Hello World Hello Python"`, the output should be `{"Hello": 2, "World": 1, "Python": 1}`.

Please note that this question requires you to handle punctuation, case sensitivity, and word frequency counting, making it a challenging problem.

### Human: Here's a Python function that uses regular expressions to remove punctuation and converts the string to lowercase for case-insensitive counting:

```python
import re

def word_freq(input_str):
    # Remove non-alphanumeric characters and convert to lower case
    cleaned_str = re.sub(r'[^\w\s]', '', input_str).lower()
    
    # Split into words and count their frequency using a dictionary comprehension
    return {
        word: cleaned_str.count(word)
        for word in set(cleaned_str.split())
    }
```

This function works as follows:

1. It first removes all non-alphanumeric characters from the string (including punctuation) using `re.sub()`.
2. Then, it converts the cleaned string to lowercase with the `lower()` method.
3. Next, it splits the cleaned and lowercased string into individual words using the `split()` method (which splits on whitespace by default).
4. Finally, it uses a dictionary comprehension to count the frequency of each unique word in the list.

Here's an example usage:

```python
print(word_freq("Hello World Hello Python"))
```

```python
{'hello': 3, 'world': 1, 'python': 1}
```
