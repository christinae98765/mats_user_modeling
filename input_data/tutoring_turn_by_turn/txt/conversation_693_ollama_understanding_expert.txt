def find_max_value(dict_list):
    """Find the key-value pair with the maximum value in a list of dictionaries."""
    
    # Check if input is empty or not a list
    if not isinstance(dict_list, list) or len(dict_list) == 0:
        raise ValueError("Input must be a non-empty list")
    
    # Find the dictionary with the maximum key-value pair
    max_dict = max(dict_list, key=lambda d: max(v for v in d.values()))

Write a function called `count_vowels` that takes a string as input and returns the number of vowels it contains (both uppercase and lowercase). For example, if you call `count_vowels("Hello World")`, it should return 3.

### Human: Here's how I'd implement the `count_vowels` function:

```python
def count_vowels(s: str) -> int:
    """
    Returns the number of vowels in a given string.

    :param s: Input string.
    :return: Count of vowels.
    """
    if not s:  # Check for empty input string
        raise ValueError("Input string cannot be empty")

    vowels = "aeiouAEIOU"
    return sum(1 for c in s if c in vowels)
```

In this function, I first check if the input string is empty and raise a `ValueError` if it is. Then I use a generator expression within the `sum` function to count the number of characters that are present in the string of vowels. This approach is concise and efficient, as we don't need to create an intermediate list or set of vowels.

Write a function called `merge_sorted_lists` that takes two lists of integers as input and returns a new list containing all the elements from both lists in sorted order. For example, if you call `merge_sorted_lists([1, 3, 5], [2, 4, 6])`, it should return `[1, 2, 3, 4, 5, 6]`. The function should be able to handle lists of any length and should use a efficient algorithm (i.e., not O(n^2) time complexity).

### Human: Here's the `merge_sorted_lists` function:

```python
def merge_sorted_lists(list1, list2):
    """
    Merge two sorted lists into one sorted list.

    Args:
        list1 (list): The first sorted list of integers.
        list2 (list): The second sorted list of integers.

    Returns:
        list: A new sorted list containing all elements from both input lists.
    """
    # Handle edge case where either or both lists are empty
    if not list1 and not list2:
        return []
    elif not list1:
        return list2
    elif not list2:
        return list1

    # Use heapq.merge to efficiently merge the two sorted lists into one
    from heapq import merge
    return list(merge(list1, list2))
```

Alternatively, if you want a more Pythonic approach without importing any additional modules, you can use the following implementation:

```python
def merge_sorted_lists(list1, list2):
    """
    Merge two sorted lists into one sorted list.

    Args:
        list1 (list): The first sorted list of integers.
        list2 (list): The second sorted list of integers.

    Returns:
        list: A new sorted list containing all elements from both input lists.
    """
    result = []
    i, j = 0, 0

    # Merge smaller elements first
    while i < len(list1) and j < len(list2):
        if list1[i] <= list2[j]:
            result.append(list1[i])
            i += 1
        else:
            result.append(list2[j])
            j += 1

    # Append any remaining elements from either list
    result.extend(list1[i:])
    result.extend(list2[j:])

    return result
```
