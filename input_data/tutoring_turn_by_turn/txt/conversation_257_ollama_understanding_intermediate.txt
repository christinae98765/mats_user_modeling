### Human: Here's a simple implementation of the `reverse_string` function:

```python
def reverse_string(input_str):
    return input_str[::-1]
```

This function uses Python's slice notation to extract characters from the string in reverse order, effectively reversing it. The `::-1` means start at the end of the string and end at position 0, move with the step -1 which means one step backwards.
