The function should return a list of nodes that have multiple edges pointing to them. For example, if the input is {1: 2, 2: 3, 3: 4, 4: 5, 2: 3}, the output should be [2, 3].

### Human: def find_duplicates(graph):
    duplicates = []
    for node in graph:
        if graph[node] in graph.values():
            if graph.count(graph[node]) > 1 and node not in duplicates:
                duplicates.append(node)
    return duplicates

### Assistant: Your code attempts to solve the problem, but it's not quite correct. Here's a medium question:

Write a function called `reverse_sentence` that takes a string as input and returns a new string with all words in reverse order. For example, if the input is "hello world", the output should be "world hello".

### Human: def reverse_sentence(sentence):
    words = sentence.split()
    reversed_words = []
    
    for word in words:
        reversed_words.insert(0, word)
        
    return ' '.join(reversed_words)

### Assistant: Good job on your attempt to solve the problem! You were close, but you used `insert(0, word)` instead of simply using `reversed_words.append(word)`. Also, since `split()` splits the string into words by default, you don't need to specify it.

Now, let's move on to a MEDIUM question. Here it is:

Write a function called `count_vowels` that takes a string as input and returns the number of vowels in the string. For example, if the input is "hello world", the output should be 3.

### Human: def count_vowels(s):
    vowels = 'aeiou'
    return sum(1 for char in s.lower() if char in vowels)
