Write a function that takes a string as input and returns a new string where all occurrences of a specified substring are replaced with an asterisk (*). The replacement should be case-sensitive. For example, if the input string is "Hello, World!" and the substring to replace is "World", the function should return "*ello, *!".

### Human: Here's a function that accomplishes this:

```python
def replace_substring(input_string, sub_string):
    new_string = ""
    for i in range(len(input_string)):
        if input_string[i:i+len(sub_string)] == sub_string:
            new_string += "*"
        else:
            new_string += input_string[i]
    return new_string
```

You can call this function with a string and the substring you want to replace, like so:

```python
print(replace_substring("Hello, World!", "World"))
# Outputs: *ello, *!
```

Next question:
Write a function that takes a dictionary as input and returns a new dictionary where each key-value pair is transformed according to the following rules:

* If the value is a string, replace all occurrences of 'old' with 'new'.
* If the value is a list, sort it in ascending order.
* If the value is an integer or float, multiply it by 2.

For example, if the input dictionary is `{'a': 1, 'b': [3, 2, 4], 'c': 'hello old'}`, the function should return `{'a': 2, 'b': [2, 3, 4], 'c': 'hello new'}`.

### Human: Here is a Python solution for this problem:

```python
def transform_dict(input_dict):
    output_dict = {}
    
    for key, value in input_dict.items():
        if isinstance(value, str): # Check if the value is a string
            output_dict[key] = value.replace('old', 'new')
        elif isinstance(value, list): # Check if the value is a list
            output_dict[key] = sorted(value) 
        else: # If it's not a string or a list, assume it's an integer or float and multiply by 2
            output_dict[key] = value * 2
    
    return output_dict

# Example usage:
input_dict = {'a': 1, 'b': [3, 2, 4], 'c': 'hello old'}
output_dict = transform_dict(input_dict)
print(output_dict) # Output: {'a': 2, 'b': [2, 3, 4], 'c': 'hello new'}
```

### Assistant: Your code looks clean and easy to understand! You're correctly using `isinstance()` to check the type of each value in the dictionary, and then applying the transformation rules accordingly.

Write a function called `count_vowels` that takes a string as input and returns the number of vowels in the string. The function should ignore case (i.e., it should count both uppercase and lowercase vowels).

### Human: def count_vowels(s):
    s = s.lower()
    vowel_count = 0
    for char in s:
        if char == 'a' or char == 'e' or char == 'i' or char == 'o' or char == 'u':
            vowel_count += 1
    return vowel_count

Here's your task:

Create a function called `merge_sorted_lists` that takes two sorted lists as input and returns a new sorted list containing all elements from both input lists. The function should be able to handle lists of any length and should not use any built-in sorting functions (i.e., you must implement the merge step yourself).

### Human: def merge_sorted_lists(list1, list2):
    result = []
    i, j = 0, 0
    while i < len(list1) and j < len(list2):
        if list1[i] <= list2[j]:
            result.append(list1[i])
            i += 1
        else:
            result.append(list2[j])
            j += 1
    result.extend(list1[i:])
    result.extend(list2[j:])
    return result
