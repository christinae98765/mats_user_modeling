    with open(file_name, 'r') as file:
        for line in file:
            words = line.lower().split()
            for word in words:
                if word not in frequency_dict:
                    frequency_dict[word] = 1
                else:
                    frequency_dict[word] += 1

### Human: Here's a Python program that takes an integer `n` as input and returns the first `n` Fibonacci numbers in a list:

```python
def fibonacci(n):
    fib_list = [0, 1]
    while len(fib_list) < n:
        next_fib = fib_list[-1] + fib_list[-2]
        fib_list.append(next_fib)
    return fib_list[:n]

# Test the function with a few examples
print(fibonacci(5))  # Output: [0, 1, 1, 2, 3]
print(fibonacci(8))  # Output: [0, 1, 1, 2, 3, 5, 8, 13]
```

This program uses a while loop to generate Fibonacci numbers until it has generated `n` of them. It starts with the list `[0, 1]`, which contains the first two Fibonacci numbers. Then, in each iteration of the loop, it calculates the next Fibonacci number by adding the last two numbers in the list and appends this new number to the end of the list. Finally, when there are `n` Fibonacci numbers in the list, it returns all `n` numbers (or fewer if `n` is less than 2).

### Human: Here's the solution:
```python
def remove_vowels(s):
    vowels = 'aeiouAEIOU'
    return ''.join([c for c in s if c not in vowels])
```
This function uses a list comprehension to iterate over each character `c` in the input string `s`. If `c` is not found in the string of vowels, it's included in the new list. Finally, the `join()` method is used to concatenate all characters in the list into a single string.

Implement a recursive function that calculates the sum of all elements in a nested dictionary (a dictionary whose values can be either numbers or other dictionaries). For example, if you call `nested_dict_sum({"a": 1, "b": {"x": 2, "y": {"z": 3}}})`, it should return `6`.

### Human: Here's a recursive function that calculates the sum of all elements in a nested dictionary:

```python
def nested_dict_sum(nested_dict):
    total = 0
    for key, value in nested_dict.items():
        if isinstance(value, dict):
            total += nested_dict_sum(value)
        else:
            total += value
    return total
```
