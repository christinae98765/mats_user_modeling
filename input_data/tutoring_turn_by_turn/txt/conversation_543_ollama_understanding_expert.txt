```python
def remove_vowels(s: str) -> str:
    """
    Removes all vowels from the input string.

    Args:
```

```python
def intersection(list1: list, list2: list) -> set:
    """
    Returns the intersection of two lists, preserving duplicates.

    Args:
        list1 (list): The first input list.
        list2 (list): The second input list.

    Returns:
        set: A set containing the elements present in both lists.
    """
    # Convert the lists to sets for efficient lookup and removal of duplicates
    set1 = set(list1)
    set2 = set(list2)

    # Find the intersection of the two sets, preserving order and duplicates from list1
    intersection_set = {elem for elem in set1 if elem in set2}

    return intersection_set

# Example usage:

list2 = [2, 2, 4]

print(intersection(list1, list2))  # Output: {2}
```

Now, here's a HARD question: Write a function that implements the merge sort algorithm to sort a list of integers. The function should take two lists as input (a left sublist and a right sublist) and return the sorted merged list. You'll need to use recursion to implement this efficiently.

### Human: Here's a Python implementation of the merge sort algorithm using recursion:

```python
def merge_sort(left, right):
    """
    Merge two sorted sublists into one.

    Args:
        left (list): The first sorted sublist.
        right (list): The second sorted sublist.

    Returns:
        list: A new sorted list containing all elements from both input lists.
    """

    # Base case: If the length of either sublist is 1 or less, return it as it's already sorted
    if len(left) <= 1 and len(right) <= 1:
        return left + right

    # Split each sublist into two halves
    mid_left = len(left) // 2
    mid_right = (len(right) - 1) // 2
    
    # Recursively sort the sublists
    left_sorted = merge_sort(left[:mid_left+1], left[mid_left+1:])
    right_sorted = merge_sort(right[:mid_right+1], right[mid_right+1:])

    # Merge the sorted sublists into one
    return merge(left_sorted, right_sorted)


def merge(list1, list2):
    """
    Merge two sorted lists into one.

    Args:
        list1 (list): The first sorted list.
        list2 (list): The second sorted list.

    Returns:
        list: A new sorted list containing all elements from both input lists.
    """

    result = []
    i, j = 0, 0

    # Compare elements from both lists and append the smaller one to the result
    while i < len(list1) and j < len(list2):
        if list1[i] <= list2[j]:
            result.append(list1[i])
            i += 1
        else:
            result.append(list2[j])
            j += 1

    # Append any remaining elements from either list
    result.extend(list1[i:])
    result.extend(list2[j:])

    return result


# Example usage:
left = [5, 3]
right = [8, 9, 1]

sorted_list = merge_sort(left, right)
print(sorted_list)  # Output: [1, 3, 5, 8, 9]
```

This implementation uses the `merge` function to combine two sorted sublists into one. The `merge_sort` function recursively splits each sublist in half until they contain only one element (the base case), and then merges them using the `merge` function

### Human: Here's a Python function to achieve this:

```python
def replace_substring(s: str, sub: str) -> str:
    """
    Replace all occurrences of a specified substring with an asterisk (*) character.

    Args:
        s (str): The input string.
        sub (str): The substring to be replaced.

    Returns:
        str: A new string where all occurrences of the substring are replaced with '*'.
    """

    # Use f-string formatting to replace the substring
    return s.replace(sub, '*')

# Example usage:
input_str = "hello world"

output_str = replace_substring(input_str, sub_str)
print(output_str)  # Output: hello *
```

This function uses Python's built-in `str.replace()` method to replace all occurrences of the specified substring with an asterisk. The result is a new string where all occurrences of the substring are replaced.

Note that this approach assumes that the substring is not empty. If you want to handle empty substrings, you can add some error checking code to the function.
