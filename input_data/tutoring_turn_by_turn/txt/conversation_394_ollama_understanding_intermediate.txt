Write a function called `parse_csv_file` that reads a CSV file and returns a dictionary where each key is a unique value in the first column and the corresponding value is a list of values from the second column. For example, if your CSV file looks like this:
```
Name,Age
John,25

Bob,20
```

Your function should ignore any rows where there's missing data (i.e., empty strings). Can you write a function that does this?

### Human: Here is the `parse_csv_file` function:

```python
import csv

def parse_csv_file(filename):
    result = {}
    
    with open(filename, 'r') as file:
        reader = csv.reader(file)
        
        for row in reader:
            if len(row) == 2 and row[0] != '' and row[1] != '':
                name = row[0]
                age = int(row[1])
                
                if name not in result:
                    result[name] = [age]
                else:
                    result[name].append(age)
                    
    return result
```

This function opens the specified file, reads it line by line using `csv.reader`, checks each row to ensure that there is no missing data and converts the age to an integer. Then it updates the corresponding dictionary entry with the new age.

Here's a MEDIUM question:

Write a function called `most_frequent_word` that takes a string as input and returns the most frequent word in the string. You can assume that words are separated by spaces, and you don't need to handle punctuation or capitalization. Can you write a function that does this?

### Human: def most_frequent_word(s):
    # Split the string into words
    words = s.split()
    
    # Create a dictionary to store word frequencies
    freq_dict = {}
    
    # Loop through each word in the list and update its frequency
    for word in words:
        if word.lower() not in freq_dict:
            freq_dict[word.lower()] = 1
        else:
            freq_dict[word.lower()] += 1
    
    # Find the maximum frequency
    max_freq = max(freq_dict.values())
    
    # Return the most frequent word(s)
    most_frequent = [word for word, freq in freq_dict.items() if freq == max_freq]
    
    return most_frequent

### Assistant: Your `most_frequent_word` function looks good! You've correctly split the input string into words, counted their frequencies using a dictionary, and returned the most frequent word(s).

Here's brief feedback:

* Your function is well-organized and easy to follow.
* However, you might want to consider handling cases where there are multiple words with the same maximum frequency. Currently, your function returns all such words, but it would be more helpful if it returned a list of tuples (word, frequency) instead.

### Human: Here is a function that counts the number of vowels in a given string:

```python
def count_vowels(s):
    s = s.lower()  # convert to lowercase for simplicity
    vowel_count = 0  # initialize counter
    for char in s:
        if char in 'aeiou':  # check if character is a vowel
            vowel_count += 1  # increment counter if it's a vowel
    return vowel_count  # return the count

# test the function
```

### Human: Here's my attempt at solving the problem:

```python
def count_words_with_vowel(input_str):
    vowels = 'aeiou'
    word_count = 0
    for word in input_str.split():
        has_vowel = False
        for char in word:
            if char.lower() in vowels:
                has_vowel = True
                break
        if has_vowel:
            word_count += 1
    return word_count
```

This function splits the input string into words, then iterates over each word to check if it contains any vowels. If a word does contain a vowel, it increments the `word_count` variable and continues with the next word. Finally, it returns the total count of words that have at least one vowel.
