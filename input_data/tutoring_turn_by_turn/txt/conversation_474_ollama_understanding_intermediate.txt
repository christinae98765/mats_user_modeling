### Assistant: Write a recursive function to flatten a nested list in Python. For example, if you have `[[1, 2], [3, [4, 5]], [6]]`, it should return `[1, 2, 3, 4, 5, 6]`.

### Human: Here is the recursive function to flatten a nested list in Python:
```python
def flatten(lst):
    result = []
    for item in lst:
        if type(item) == list:
            result.extend(flatten(item))
        else:
            result.append(item)
    return result
```
This function iterates over each item in the input list. If an item is a list itself, it recursively calls `flatten` on that sublist and appends the flattened elements to `result`. If an item is not a list (i.e., it's a single element), it simply appends it to `result`. The function returns the final result, which contains all elements from the input list flattened into a single list.

Write a function that takes a string as input and returns the number of vowels in the string. For example, if you call `count_vowels("hello")`, it should return 2. You can use a while loop or other iterative approach.

### Human: def count_vowels(s):
    vowel_count = 0
    i = 0
    while i < len(s):
        char = s[i]
        if char.lower() in 'aeiou':
            vowel_count += 1
        i += 1
    return vowel_count

### Human: def merge_lists(list1, list2):
    if not list1 or not list2:
        return list1 + list2
    elif list1[0] < list2[0]:
        return [list1[0]] + merge_lists(list1[1:], list2)
    else:
        return [list2[0]] + merge_lists(list1, list2[1:])
