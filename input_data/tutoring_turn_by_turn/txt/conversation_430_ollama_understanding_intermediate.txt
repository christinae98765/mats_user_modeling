Implement a Python program that uses the `socket` library to create a simple TCP server and client. Note: You can use any relevant error handling or exception handling as needed.

### Human: Here is the Python program that implements a simple TCP server and client using the `socket` library:

```python
import socket

# Server code
def start_server():
    # Create a socket object
    serversocket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

    # Bind the socket to a address and port
    host = '127.0.0.1'
    port = 12345
    serversocket.bind((host, port))

    # Listen for connections
    serversocket.listen(5)
    print('Server is listening...')

    while True:
        # Accept connection from client
        conn, addr = serversocket.accept()
        print('Connected to: ', addr)

        # Receive message from client
        data = conn.recv(1024)
        if not data:
            break

        # Reverse the received message and send it back to client
        reversed_data = data[::-1]
        conn.sendall(reversed_data)

    # Close the connection
    conn.close()
    serversocket.close()

# Client code
def start_client():
    # Create a socket object
    clientsocket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

    # Connect to server
    host = '127.0.0.1'
    port = 12345
    clientsocket.connect((host, port))

    # Send message to server
    message = input("Enter a message: ")
    clientsocket.sendall(message.encode())

    # Receive response from server and print it
    data = clientsocket.recv(1024)
    print('Received from server:', data.decode())

    # Close the connection
    clientsocket.close()

# Main code
if __name__ == "__main__":
    import threading

    # Start server in a separate thread
    t1 = threading.Thread(target=start_server)
    t1.daemon = True  # Set as daemon so it exits when main thread finishes
    t1.start()

    # Wait for client to connect and send message
    input("Press Enter to send message...")

    # Start client in the same process
    start_client()
```

Note that this code will create two sockets, one for the server and one for the client. The server will listen for connections on port 12345, while the client connects to the server at the same address and port. When a message is sent from the client, the server receives it, reverses it, and sends it back to the client, which prints the response

Here's a MEDIUM question:

Write a Python function that takes a string as input and returns a new string with all vowels removed from it. For example, if the input is "hello world", the output should be "hll wrld". You can assume the input only contains alphabetic characters (no numbers or special characters).

### Human: ```python
def remove_vowels(s):
    """
    Removes all vowels from a given string.
    
    Parameters:
    s (str): The input string to remove vowels from.
    
    Returns:
    str: A new string with all vowels removed.
    """
    # Define the set of vowels in lowercase and uppercase
    vowels = set('aeiouAEIOU')
    
    # Use a list comprehension to create a new list where each character is included if it's not a vowel
    no_vowels_list = [char for char in s if char not in vowels]
    
    # Join the characters back into a string using an f-string
    return ''.join(no_vowels_list)

# Test the function with some example inputs
print(remove_vowels("hello world"))  # Output: "hll wrld"

```
