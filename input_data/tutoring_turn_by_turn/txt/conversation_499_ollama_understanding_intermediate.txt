Write a function that takes two lists as input and returns their intersection (i.e., the elements they have in common). The output should be a list of these common elements, without duplicates.

### Human: def intersect_lists(list1, list2):
    if not isinstance(list1, list) or not isinstance(list2, list):
        raise ValueError("Both inputs must be lists")

    set1 = set(list1)
    set2 = set(list2)

    intersection = set1.intersection(set2)

    return list(intersection)

### Assistant: Your `intersect_lists` function is clear and concise! It checks if both inputs are lists, converts them to sets for efficient lookup, finds the intersection of these sets using the `intersection` method, and returns this intersection as a list.

Write a function that takes a string as input and returns the number of unique words in the string. For example, if the input is "hello world hello", the output should be 2.

(Note: You can assume the input string will not contain any punctuation.)

### Human: Here's a solution for this problem:

```python
def count_unique_words(input_string):
    # Split the input string into words (assuming space-separated)
    words = input_string.split()
    
    # Convert the list of words to a set, which automatically removes duplicates
    unique_words = set(words)
    
    # Return the number of unique words in the set
    return len(unique_words)

# Example usage:
input_str = "hello world hello"
num_unique_words = count_unique_words(input_str)
print(num_unique_words)  # Output: 2
```

This function first splits the input string into individual words using the `split()` method. It then converts this list of words to a set, which automatically removes any duplicate words because sets only store unique elements. Finally, it returns the number of unique words in the set by getting its length using `len()`.

### Assistant: Your `count_unique_words` function is a great example of how to use sets to efficiently remove duplicates from a list.
