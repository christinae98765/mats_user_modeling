Write a function called `find_duplicates` that takes a list of integers as input and returns a dictionary where the keys are unique numbers from the list and the values are lists of indices at which each number appeared in the original list.

For example, if you have the list `[1, 2, 3, 4, 5, 2, 3]`, your function should return `{1: [0], 2: [1, 5], 3: [2, 6], 4: [3], 5: [4]}`.

This one requires some creative thinking and dictionary manipulation. Give it a shot!

### Human: Here's my solution:
```python
def find_duplicates(nums):
    result = {}
    for i, num in enumerate(nums):
        if num not in result:
            result[num] = [i]
        else:
            result[num].append(i)
    return result
```
This function iterates over the input list `nums` using `enumerate`, which provides both the index `i` and the value `num`. For each number, it checks if it already exists in the dictionary `result`. If not, it adds a new key-value pair with the current index as the value. If it does exist, it appends the current index to the list of values for that key. Finally, it returns the resulting dictionary.
