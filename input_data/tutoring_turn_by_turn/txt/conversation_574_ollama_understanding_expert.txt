Here's a MEDIUM question:

Write a function that takes a string as input and returns the number of words in the string. For example, if the input is "Hello World, this is a test", the output should be 5 (counting each word separately). How would you handle punctuation at the end of a word?

### Human: Here's how I'd approach this problem:

```python
import re

def count_words(s: str) -> int:
    """
    Count the number of words in a given string, ignoring trailing punctuation.

    Args:
        s (str): Input string to be processed.

    Returns:
        int: Number of words in the input string.
    """

    # Remove leading and trailing whitespace
    s = s.strip()
    
    # Split the string into words using one or more non-word characters as separators
    words = re.split(r'\W+', s)
    
    # Filter out empty strings (which can result from consecutive punctuation marks)
    words = [word for word in words if word]

    return len(words)
```

This function works by first removing any leading or trailing whitespace, then splitting the string into words using regular expressions. The `\W+` pattern matches one or more non-word characters, which serves as a delimiter between words.

To handle punctuation at the end of a word, we use a list comprehension to filter out empty strings that can result from consecutive punctuation marks.

This approach is concise and easy to understand, making it suitable for both developers who are familiar with Python and those who are new to the language.

### Assistant: Your code is well-structured and easy to follow! You've handled trailing whitespace correctly by using `strip()`, and your use of regular expressions to split the string into words is a good approach.
