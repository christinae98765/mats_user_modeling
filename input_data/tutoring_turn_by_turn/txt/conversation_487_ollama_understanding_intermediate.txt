### Human: def longest_substring(s):
    start = 0
    max_length = 0
    unique_chars = {}
    
    for i in range(len(s)):
        if s[i] in unique_chars and start <= unique_chars[s[i]]:
            start = unique_chars[s[i]] + 1

```python
def count_words(filename):
    word_count = {}
    with open(filename, 'r') as file:
        for line in file:
            words = line.split()
            for word in words:
                if word.lower() not in word_count:
                    word_count[word.lower()] = 1
                else:
                    word_count[word.lower()] += 1
    for word, count in word_count.items():
        print(f'{word}: {count}')

count_words('example.txt')
```

This code defines a function `count_words` that takes the filename as input. It opens the file and reads it line by line. For each line, it splits the string into words using `split()`, then for each word, it checks if it's already in the dictionary. If not, it adds it with a count of 1; otherwise, it increments the existing count. Finally, it prints out each unique word and its frequency.

This code will print out all unique words from the file along with their frequencies. Note that this code is case-insensitive, so 'Word' and 'word' are considered the same word.

### Assistant: Your code is well-structured and effectively uses a dictionary to count the frequency of each word in the file. It's also nice that you've made it case-insensitive.
