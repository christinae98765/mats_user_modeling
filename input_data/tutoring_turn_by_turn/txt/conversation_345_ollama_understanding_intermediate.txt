Write a function that reads a text file containing a list of student names and their corresponding grades. The function should return a dictionary where each key is a student name and the value is a dictionary with two keys: 'average' and 'std_dev'. The average grade for each student should be calculated, as well as the standard deviation of their grades.

The text file will have the following format:
```
John Doe 85
Jane Doe 90
John Doe 78
...
```
You can assume that the input file is properly formatted and contains only valid data.

### Human: def calculate_student_grades(filename):
    with open(filename, 'r') as f:
        lines = [line.strip().split() for line in f.readlines()]

    student_grades = {}

    for line in lines:
        name = line[0] + ' ' + line[1]
        grade = int(line[2])

        if name not in student_grades:
            student_grades[name] = {'grades': [grade]}
        else:
            student_grades[name]['grades'].append(grade)

    for name, data in student_grades.items():
        grades = data['grades']
        average = sum(grades) / len(grades)
        variance = sum((x - average) ** 2 for x in grades) / (len(grades) - 1)
        std_dev = variance ** 0.5
        student_grades[name]['average'] = round(average, 2)
        student_grades[name]['std_dev'] = round(std_dev, 2)

    return student_grades

### Human: Here is my solution:

```python
def reverse_string(s):
    # Use slicing to get the characters in reverse order
    return s[::-1]
```

This function uses Python's slicing feature, which allows you to extract a subset of characters from a string. By starting at the end of the string and moving backwards to the beginning (`s[::-1]`), we can easily reverse the string.

Example usage:
