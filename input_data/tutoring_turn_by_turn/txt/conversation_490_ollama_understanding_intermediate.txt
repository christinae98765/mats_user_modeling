Write a Python function that takes a dictionary `d` as input and returns a new dictionary with all keys capitalized. For example, if `d = {'a': 1, 'b': 2}`, the function should return `{A: 1, B: 2}`.

What's your attempt?

### Human: ```python
def capitalize_keys(d):
    result = {}
    for key in d:
        value = d[key]
        new_key = key.upper()
        result[new_key] = value
    return result
```

This function iterates over each key-value pair in the input dictionary `d`. It converts each key to uppercase using the `upper()` method, and stores this new key along with its original value in a new dictionary `result`. Finally, it returns `result`, which contains all keys capitalized.

Write a Python function that takes two lists `l1` and `l2` as input, where each list contains strings representing file paths. The function should return a dictionary where each key is a directory name from the file paths in `l1`, and the corresponding value is a list of files in that directory. For example, if `l1 = ['/home/user/file1.txt', '/home/user/file2.txt', '/etc/config.conf']` and `l2 = ['/usr/bin/executable.sh', '/home/user/file3.txt', '/var/log/error.log']`, the function should return `{'/home/user': ['file1.txt', 'file2.txt', 'file3.txt'], '/etc': ['config.conf']}`.

Note: You'll need to handle nested directories, and you can't assume that all directory names are present in both lists.

### Human: Here is a Python function that accomplishes this:

```python
def organize_files(l1, l2):
    # Initialize an empty dictionary to store the organized files
    organized = {}

    # Iterate over each file path in the first list
    for file in l1:
        # Split the file path into components (directories and file name)
        components = file.split('/')
        
        # Get the directory name by joining all components except the last one
        dir_name = '/'.join(components[:-1])
        
        # If this directory is not already in our dictionary, add it with an empty list
        if dir_name not in organized:
            organized[dir_name] = []
        
        # Add the file to its corresponding directory's list
        organized[dir_name].append(components[-1])

    # Now do the same for the second list of files
    for file in l2:
        components = file.split('/')
        dir_name = '/'.join(components[:-1])
        if dir_name not in organized:
            organized[dir_name] = []
        organized[dir_name].append(components[-1])

    return organized

# Example usage:
l1 = ['/home/user/file1.txt', '/home/user/file2.txt', '/etc/config.conf']
l2 = ['/usr/bin/executable.sh', '/home/user/file3.txt', '/var/log/error.log']

print(organize_files(l1, l2))
```

This function uses a simple loop to iterate over each file path in both input lists. It splits each file path into its components (directories and file name), gets the directory name by joining all but the last component, and adds the file to its corresponding directory's list in the output dictionary. If a directory is not already present in the dictionary, it is added with an empty list.

Write a Python function that takes an integer `n` as input and returns the first `n` Fibonacci numbers. For example, if `n = 5`, the function should return `[0, 1, 1, 2, 3]`.

Can you write this function?

### Human: def fibonacci(n):
    """
    Returns a list of the first n Fibonacci numbers.
    
    The Fibonacci sequence is a series of numbers where each number is the sum of the two preceding ones. 
    Starting with 0 and 1, the sequence goes: 0, 1, 1, 2, 3, 5, 8, 13, ...
    
    Parameters:
    n (int): The number of Fibonacci numbers to generate.
    
    Returns:
    list: A list of the first n Fibonacci numbers.
    """
    fib_sequence = [0, 1]
    while len(fib_sequence) < n:
        next_fib = fib_sequence[-1] + fib_sequence[-2]
        fib_sequence.append(next_fib)
    return fib_sequence[:n]

# Test the function
print(fibonacci(5))  # Output: [0, 1, 1, 2, 3]

### Assistant: Your code looks good! You're correctly generating the Fibonacci sequence using a simple loop and appending each new number to the end of the list.

However, you might want to consider adding some error handling for cases where `n` is less than 2, since the Fibonacci sequence requires at least two numbers to be generated.

Implement a Python function that takes a dictionary of employee data and returns a list of all employees who are eligible for a company-wide promotion. An employee is eligible if they have been working at the company for more than 5 years, have a salary above $100,000, and have a performance rating of "Excellent" or higher.

```python
employees = {
    'John': {'salary': 120000, 'years_of_service': 7, 'performance_rating': 'Outstanding'},
    'Jane': {'salary': 90000, 'years_of_service': 3, 'performance_rating': 'Good'},
    'Bob': {'salary': 150000, 'years_of_service': 10, 'performance_rating': 'Excellent'}
}
```

Can you write a function to solve this problem?

### Human: def eligible_for_promotion(employees):
    promotion_eligible = []
    for employee in employees:
        if employees[employee]['salary'] > 100000 and \
           employees[employee]['years_of_service'] > 5 and \
           (employees[employee]['performance_rating'] == 'Excellent' or
            employees[employee]['performance_rating'] == 'Outstanding'):
            promotion_eligible.append(employee)
    return promotion_eligible

print(eligible_for_promotion(employees))
# Output: ['John', 'Bob']

Here's your next question:

Write a Python function that takes a string `s` as input and returns the first occurrence of a word that appears at least twice in the string. If no such word exists, return an empty string.

For example, if `s = 'apple banana apple'`, the function should return `'apple'`.

### Human: def find_repeated_word(s):
    words = s.split()
    repeated_words = set()

    for word in words:
        if words.count(word) > 1 and word not in repeated_words:
            repeated_words.add(word)
            return word

    return ''
