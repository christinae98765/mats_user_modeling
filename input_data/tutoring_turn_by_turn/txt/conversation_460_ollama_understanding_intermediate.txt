Write a function that takes two lists of integers as input and returns a new list containing the elements that are common to both lists. The order of the elements in the output list should match the order they appear in the second input list.

### Human: Here is the solution:
```python
def common_elements(list1, list2):
    return [x for x in list2 if x in list1]
```
This function uses a list comprehension to iterate over each element `x` in `list2`. For each `x`, it checks if `x` is present in `list1` using the `in` operator. If `x` is found, it is included in the output list.

This solution assumes that the input lists contain only integers and that there are no duplicate elements within a single list.

### Assistant: Your function is clear and concise, but it has a time complexity of O(n^2) because the `in` operator is used in the list comprehension. This can be optimized to O(n) using a set.

Write a function that takes a nested dictionary as input (where each value is either an integer or another dictionary), and returns the sum of all integers found within the dictionary, without traversing any dictionaries more than once.

### Human: def sum_integers(nested_dict):
    total = 0
    visited = set()
    
    def helper(dictionary):
        nonlocal total, visited
        
        for key, value in dictionary.items():
            if key not in visited:
                visited.add(key)
                
                if isinstance(value, int):
                    total += value
                    
                elif isinstance(value, dict) and id(value) not in [id(x) for x in visited]:
                    helper(value)
                    
    helper(nested_dict)
    
    return total

### Assistant: Your recursive function is a good approach to solve this problem.

However, you're using `id(value)` which is not necessary here because dictionaries are hashable and can be added to sets directly. Also, your condition for visiting each dictionary only once could be simplified by checking if the key is already in the set before adding it.

Implement a recursive function that traverses a nested dictionary and returns the maximum value found. The dictionary may contain integer values, as well as other dictionaries with their own values. Note: You should not modify the original dictionary.

### Human: def max_value(nested_dict):
    def helper(d):
        if isinstance(d, int):
            return d
        else:
            return max(value for value in d.values() if isinstance(value, (int, dict)))
    
    if isinstance(nested_dict, int):
        return nested_dict
    else:
        return max(helper(value) for value in nested_dict.values())

# Test the function

    "b": 2,
    "c": {

            "g": 5,
            "h": 6
        }
    },
    "i": {
        "j": 7,
        "k": 8
    }
