### Human: Here's my solution:

```python
def sum_nested_list(lst):
    """Recursively calculates the sum of all integers in a nested list."""
    
    total = 0
    
    for item in lst:
        if isinstance(item, int):  # Check if item is an integer
            total += item
        elif isinstance(item, list):  # If item is a list, recurse into it
            total += sum_nested_list(item)
    
    return total

# Example usage:
nested_list = [1, 2, [3, 4, [5, 6]], 7]
print(sum_nested_list(nested_list))  # Output: 28
```

This solution uses recursion to traverse the nested list and calculate the sum of all integers. The `isinstance()` function checks if an item is either an integer or a list, allowing us to handle both cases correctly.

def find_target(s, target):

This function tries to find the index of `target` within `s` using `str.index()`. If a match is found, it returns the index. Otherwise, it catches the `ValueError` exception raised by `str.index()` when no match is found and returns `-1`.

def find_target(s, target):
