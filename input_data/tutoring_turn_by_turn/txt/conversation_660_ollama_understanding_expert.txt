### Assistant: Write a recursive function in Python to flatten a nested list of integers. For example, if the input is [1, [2, 3], [4, [5, [6]]]], the output should be [1, 2, 3, 4, 5, 6].

### Human: Here's a recursive function that flattens a nested list of integers:

```python
def flatten(nested_list):
    result = []
    for item in nested_list:
        if isinstance(item, list):
            result.extend(flatten(item))
        else:
            result.append(item)
    return result

# Example usage:
nested_list = [1, [2, 3], [4, [5, [6]]]]
print(flatten(nested_list))  # Output: [1, 2, 3, 4, 5, 6]
```

However, for a more Pythonic approach, we can use list comprehension:

```python
def flatten(nested_list):
    return [item for sublist in nested_list for item in (flatten(sublist) if isinstance(sublist, list) else [sublist])]

# Example usage:
nested_list = [1, [2, 3], [4, [5, [6]]]]
print(flatten(nested_list))  # Output: [1, 2, 3, 4, 5, 6]
```

In both examples, we iterate through each item in the nested list. If an item is a list itself, it's recursively flattened and extended to the result. Otherwise, the item is directly appended to the result.

### Human: Here's the function to calculate average grades for each student:
```python
def avg_student_grades(students_grades):
    """Calculate average grade for each student."""
    
    # Use dictionary comprehension with weighted sum and division
    return {
        student: sum(grades) / len(grades)
        for student, grades in students_grades.items()
    }
```
This function works by iterating over the input dictionary using a dictionary comprehension. For each student and their list of grades, it calculates the average grade by taking the sum of all grades and dividing by the number of grades.

Now, let's tackle a MEDIUM question:

Write a function that takes a string as input and returns the longest substring without repeating characters. For example, if the input is "abcabcbb", the output should be "abc".

### Human: Here's a Python solution using a sliding window approach with a set to keep track of unique characters:
```python
def longest_substring(s: str) -> str:
    """
    Returns the longest substring without repeating characters.

    :param s: Input string
    :return: Longest substring without repeating characters
    """
    if not isinstance(s, str):
        raise ValueError("Input must be a string")

    char_set = set()
    left = 0
    max_length = 0
    max_substring = ""

    for right in range(len(s)):
        while s[right] in char_set:
            # If character is already in the set, remove leftmost character from the set and update left pointer
            char_set.remove(s[left])
            left += 1

        char_set.add(s[right])

        if right - left + 1 > max_length:
            # Update maximum length and substring
            max_length = right - left + 1
            max_substring = s[left:right+1]

    return max_substring
```
This function iterates over the input string using a sliding window approach, where `left` and `right` pointers represent the start and end of the current substring. It uses a set to keep track of unique characters in the current substring. When it encounters a repeating character, it removes the leftmost character from the set and updates the `left` pointer.

Note that this function has a time complexity of O(n), where n is the length of the input string. The space complexity is also O(n) due to the use of a set to store unique characters.

### Assistant: Your code is clear and well-structured, using a set to efficiently keep track of unique characters in the current substring.

```python
def print_first_even_numbers(n):
    """
    Prints the first n even numbers.

    Args:
        n (int): The number of even numbers to print.
    """
    even_numbers = [i for i in range(2, 2*n+1, 2)]

```

This program uses a list comprehension to generate the first `n` even numbers.
