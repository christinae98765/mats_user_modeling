Here's a MEDIUM question:

Write a function `find_palindromes` that takes a list of strings as input and returns a new list containing only the strings that are palindromes (i.e., they read the same forwards and backwards).

### Human: ```python
def find_palindromes(strings):
    """
    Returns a new list containing only the palindromic strings from the input.

    Args:
        strings (list): A list of strings to check for palindromes.

    Returns:
        list: A new list containing only the palindromic strings.
    """
    # Initialize an empty list to store the palindrome strings
    palindromes = []

    # Iterate over each string in the input list
    for s in strings:
        # Check if the string is equal to its reverse
        if s == s[::-1]:
            # If it's a palindrome, add it to our list of palindromes
            palindromes.append(s)

    # Return the list of palindromic strings
    return palindromes

# Example usage:
strings = ["radar", "hello", "level", "python"]
palindrome_strings = find_palindromes(strings)
print(palindrome_strings)  # Output: ['radar', 'level']
```

Here's a MEDIUM question:

Write a function `remove_duplicates` that takes a list of strings as input and returns a new list containing only the unique strings. The order of the strings in the output list should match the order they appeared in the input list. For example, if the input is `["hello", "world", "hello", "abc"]`, the output should be `["hello", "world", "abc"]`.

### Human: Here's a function that uses a for loop and a set to keep track of unique strings:
```python
def remove_duplicates(input_list):
    seen = set()
    output = []
    for string in input_list:
        if string not in seen:
            seen.add(string)
            output.append(string)
    return output

print(remove_duplicates(["hello", "world", "hello", "abc"]))
```
This function iterates over the input list, adding each string to the `seen` set. If a string is already in the `seen` set, it skips that iteration of the loop. Otherwise, it adds the string to the output list and the `seen` set.

Alternatively, we could use Python's built-in `dict.fromkeys()` method to simplify the function:
```python
def remove_duplicates(input_list):
    return list(dict.fromkeys(input_list))

print(remove_duplicates(["hello", "world", "hello", "abc"]))
```

### Assistant: Your implementation of `remove_duplicates` is good! You've used a set to efficiently keep track of unique strings, which is a great approach.

However, you've also provided an alternative using `dict.fromkeys()`, which is even simpler and more idiomatic Python. Well done!

Now, here's a HARD question:

Write a function `count_anagrams` that takes two lists as input: a list of words and a list of strings. The function should return the number of strings in the second list that are anagrams of any word in the first list. For example, if the inputs are `[["cat", "dog"], ["tac", "god", "abc"]]`, the output should be `2`.

Note: An anagram is a string formed by rearranging the characters of another string.

### Human: def count_anagrams(words, strings):
    anagrams = 0
    for word in words:
        sorted_word = ''.join(sorted(word))
        for string in strings:
            if sorted(string) == sorted_word:
                anagrams += 1
    return anagrams

# Example usage:
words = ["cat", "dog"]
strings = ["tac", "god", "abc"]
print(count_anagrams(words, strings))  # Output: 2
