Now, write a function that takes two dictionaries, `dict1` and `dict2`, as input. The function should return a new dictionary containing all key-value pairs from both input dictionaries, with the condition that if a key exists in both dictionaries, its value is taken from `dict2`.

### Human: Here's the code for this task:

```python
def combine_dicts(dict1, dict2):
    # Create an empty dictionary to store the combined result
    combined = {}
    
    # First, add all key-value pairs from dict1 into combined
    for key in dict1:
        combined[key] = dict1[key]
        
    # Then, update combined with key-value pairs from dict2, overwriting any existing keys
    for key in dict2:
        combined[key] = dict2[key]
    
    return combined

# Test the function
```

Write a function that takes two lists as input and returns a new list containing all elements from both lists, but with any duplicates removed and the original order of each element preserved.

### Human: Here is a solution using Python sets to remove duplicates while preserving the original order:
```python
def merge_lists(list1, list2):
    return [x for x in set(list1 + list2) if (list1 + list2).index(x) <= min(len(list1), len(list2))]
```
This function works by first combining both input lists into one and adding them to a set. This automatically removes any duplicates since sets only store unique elements. Then, it uses a list comprehension to filter the combined list, keeping only the elements that were present in the original lists (i.e., not in the order they appear in the combined list). The `index()` method is used to find the position of each element in the combined list, and we keep only those elements whose position is less than or equal to the length of the shorter input list. This preserves the original order of each element.

### Assistant: Your function `merge_lists()` is a creative and effective way to merge two lists while preserving order and removing duplicates.
