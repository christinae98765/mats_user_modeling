Write a function called `count_vowels` that takes a string as input and returns the number of vowels in it. For example, if you call `count_vowels("hello")`, it should return `2`.

### Human: Here's a simple function to count the vowels in a given string:

```python
def count_vowels(s):
    vowels = "aeiou"
    count = 0
    for char in s:
        if char.lower() in vowels:

    return count
```

Implement a function called `parse_json` that takes a string containing JSON data as input, parses it into a Python dictionary, and then returns a new dictionary with only the fields that have changed since the last time the function was called. If this is the first call to the function, or if no previous values were stored, return an empty dictionary.

Assume that the `json` module is imported and available for use.

### Human: ```python
import json

changed_values = {}  # store previously parsed JSON data

def parse_json(json_string):
    global changed_values  # modify this variable outside function
    
    if json_string == '':  # handle empty string input
        return {}
    
    try:
        new_dict = json.loads(json_string)  # parse JSON into a Python dictionary
        
        # compare current and previous dictionaries to find changed values
        changed_values = {key: value for key, value in new_dict.items()
                          if key not in changed_values or value != changed_values[key]}
        
    except ValueError:
        print("Invalid JSON string")
    
    return changed_values

# test the function with some example data
json_string1 = '{"name": "John", "age": 30}'
print(parse_json(json_string1))  # prints: {'name': 'John', 'age': 30}

json_string2 = '{"name": "Jane", "age": 31}'  # changed age, added new key "city"
new_values = parse_json(json_string2)
print(new_values)  # prints: {'age': 31, 'city': 'value'}
```

This code uses a global variable `changed_values` to store the previously parsed JSON data. The function parses the input string into a Python dictionary using the `json.loads()` method and compares it with the stored values to find changed fields. If this is the first call or if no previous values were stored, an empty dictionary is returned.

For a MEDIUM question:

Write a function called `remove_duplicates` that takes a list of strings as input and returns a new list with all duplicate strings removed. For example, if you call `remove_duplicates(["hello", "world", "hello", "foo"])`, it should return `["hello", "world", "foo"]`.

### Human: Here's the function to remove duplicates from a list of strings:

```python
def remove_duplicates(input_list):
    """
    Removes duplicate strings from a given list.

    Args:
        input_list (list): A list of strings.

    Returns:
        list: A new list with all duplicate strings removed.
    """
    # Create an empty dictionary to store unique strings as keys
    unique_strings = {}

    # Iterate over the input list and add each string to the dictionary
    for string in input_list:
        # Since dictionaries cannot have duplicate keys, this will automatically remove duplicates
        unique_strings[string] = None

    # Return a list of the dictionary's keys (i.e., the unique strings)
    return list(unique_strings.keys())

# Example usage:
print(remove_duplicates(["hello", "world", "hello", "foo"]))  # Output: ["hello", "world", "foo"]
```

### Assistant: Your implementation is clever and efficient! Using a dictionary to store unique strings as keys is a great way to remove duplicates from a list.

```python
def merge_sorted_lists(list1, list2):
    result = []
    i = j = 0
    
    while i < len(list1) and j < len(list2):
        if list1[i] <= list2[j]:
            result.append(list1[i])
            i += 1
        else:
            result.append(list2[j])
            j += 1
            
    # Append any remaining elements from the two lists
    while i < len(list1):
        result.append(list1[i])
        i += 1
        
    while j < len(list2):

        j += 1
        
    return result
```
