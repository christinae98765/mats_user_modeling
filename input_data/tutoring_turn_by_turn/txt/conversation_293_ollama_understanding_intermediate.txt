Now, let's move on to a MEDIUM question:

Write a function that takes a string as input and returns the longest substring without any repeating characters. For example, if the input is "abcabcbb", the output should be "abc".

### Human: def longest_substring(s):
    max_length = 0
    start_index = 0
    used_chars = {}

    for i in range(len(s)):
        if s[i] in used_chars and used_chars[s[i]] >= start_index:
            start_index = used_chars[s[i]] + 1
        used_chars[s[i]] = i
        max_length = max(max_length, i - start_index + 1)

    return s[start_index:start_index + max_length]

# Test the function
print(longest_substring("abcabcbb"))  # Output: "abc"
print(longest_substring("bbbbb"))     # Output: "b"
print(longest_substring("pwwkew"))    # Output: "wke"

### Assistant: Your `longest_substring` function is a great attempt at solving this problem! You've used a clever approach with the `used_chars` dictionary to keep track of the indices of each character.

However, your implementation has some redundancy - you're updating `max_length` inside the loop for every iteration, even if the maximum length hasn't changed. This could be optimized by only updating it when the new substring is actually longer than the previous one.
