Write a function called `double_numbers` that takes a list of numbers as input and returns a new list with each number doubled. For example, if you pass it `[1, 2, 3]`, it should return `[2, 4, 6]`.

Implement a recursive function called `count_vowels` that takes a string as input and returns the number of vowels in the string. For example, if you pass it `"hello world"`, it should return 3 (because there are three vowels: e, o, and o).

### Human: Here's the recursive function to count the vowels:

```python
def count_vowels(s):
    if len(s) == 0:
        return 0
    elif s[0].lower() in 'aeiou':
        return 1 + count_vowels(s[1:])
    else:
        return count_vowels(s[1:])
```

This function works by checking the first character of the string. If it's a vowel, it counts it and moves on to the rest of the string. If it's not a vowel, it just moves on to the rest of the string without counting it. It keeps doing this until there are no more characters left in the string, at which point it returns 0 (the base case).

### Assistant: Your recursive function to count vowels is correct! You correctly identified the base case and used the `s[0].lower()` trick to make the comparison case-insensitive.

### Human: class BankAccount:

        self.balance = 0
        self.account_number = account_number

    def deposit(self, amount):
        self.balance += amount

    def withdraw(self, amount):
        if amount > self.balance:
            raise ValueError("Insufficient funds")
        else:

Write a function called `find_substring` that takes two strings as input: a large string and a small substring. It should return the index of the first occurrence of the substring in the large string, or -1 if it's not found. For example, if you pass it `"hello world"`, `"world"`, it should return 6 (because "world" starts at index 6).

### Human: Here is a function that meets the requirements:

```python
def find_substring(big_string, small_string):
    try:
        return big_string.index(small_string)
    except ValueError:
        return -1
```

This function uses Python's built-in `index()` method of strings to find the index of the first occurrence of the substring. If it can't be found (which raises a `ValueError`), it catches that exception and returns `-1`.

### Assistant: Your `find_substring` function is concise and effectively uses Python's built-in `index()` method.

Write a function called `reverse_list` that takes a list of strings as input and returns a new list with the strings in reverse order. For example, if you pass it `["hello", "world", "abc"]`, it should return `["abc", "world", "hello"]`.

### Human: def reverse_list(input_list):
    try:
        # Using slicing to get all elements in the list and then reversing their order
        reversed_list = input_list[::-1]
        return reversed_list
    except Exception as e:
        print("An error occurred: ", str(e)) 

# Example usage
print(reverse_list(["hello", "world", "abc"]))  # Output: ["abc", "world", "hello"]
