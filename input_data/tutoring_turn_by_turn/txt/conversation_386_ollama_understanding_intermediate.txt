### Human: Here's a simple function that uses a loop to count the number of words with at least 3 characters:

```python
def count_words(s):
    word_count = 0
    for word in s.split():
        if len(word) >= 3:
            word_count += 1
    return word_count
```

This function splits the input string into a list of words using the `split()` method. It then iterates over each word, checking its length. If the word has at least 3 characters (including spaces), it increments the `word_count`. Finally, it returns the total count of words that meet this condition.
