```python
def name_frequency(file_name):
    frequency_dict = {}
    
    with open(file_name, 'r') as file:
        for line in file:
            words = line.strip().split()
            
            # Create a set to store unique words and their counts
            word_counts = set()
            for i, word in enumerate(words):
                if len(word) > 1:  # Ignore single character names
                    count = frequency_dict.get(word.lower(), 0)
                    if (word.lower(), i) not in word_counts:
                        word_counts.add((word.lower(), i))
                        frequency_dict[word.lower()] = count + 1
                
    return frequency_dict
```
This function opens the file, reads it line by line, splits each line into words, and counts the occurrence of each word. It ignores case differences and single character names. The function returns a dictionary where each key is a name from the file and its corresponding value is the number of times it appears in the file.

### Human: Here's a solution to the problem:

```python
def manual_length(s):
    length = 0
    for char in s:
        length += 1
    return length
```

This function works by initializing a variable `length` to 0. Then, it iterates over each character in the string `s`. For each character, it increments the `length` counter by 1. After iterating over all characters, it returns the final value of `length`, which is the length of the input string.

This solution has a time complexity of O(n), where n is the length of the input string, because it needs to iterate over each character once. The space complexity is O(1) because it only uses a constant amount of extra memory to store the `length` variable and the loop variable `char`.
