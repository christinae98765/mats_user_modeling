### Assistant: Write a recursive function to flatten a nested list of integers in Python, such that `[1, [2, 3], [4, [5, 6]]]` is flattened into `1, 2, 3, 4, 5, 6`.

```python
def flatten(lst):
    """
    Recursively flattens a nested list of integers.

    Args:
        lst: A list that may contain integers and/or other lists.

    Returns:
        A new list containing all integers from the input.
    """
    result = []
    
    # Iterate over each element in the list
    for elem in lst:
        if isinstance(elem, list):
            # If it's a list itself, recursively flatten it and extend the result
            result.extend(flatten(elem))
        else:
            # If it's an integer, just append it to the result
            result.append(elem)
    
    return result

# Example usage:
nested_list = [1, [2, 3], [4, [5, 6]]]
print(flatten(nested_list))  # Output: [1, 2, 3, 4, 5, 6]
```

Note that this solution uses a simple recursive approach. It's worth noting that for very deep or large nested lists, a non-recursive approach (e.g., using `itertools.chain` and generators) might be more efficient in terms of memory usage.

Implement a function that takes two lists of integers as input and returns a new list containing only the elements that appear in both lists, but with their values doubled. For example, if you have `list1 = [1, 2, 3]` and `list2 = [2, 4, 6]`, your function should return `[2 * 2, 2 * 4, 2 * 6]`.

### Human: Here's a Python function that does what you asked for:
```python
def double_common_elements(list1, list2):
    """
    Return a new list containing elements from both input lists, doubled.
    
    :param list1: The first list of integers
    :param list2: The second list of integers
    :return: A list with common elements doubled
    """
    return [x * 2 for x in set(list1) & set(list2)]
```
This function uses a set intersection to find the common elements between `list1` and `list2`, then doubles each one using a list comprehension. The `set()` conversion is used to ensure that we're only processing unique elements, so the result won't contain any duplicates.

Write a function that takes a list of strings as input and returns a new list containing only the strings that start with a vowel. For example, if you have `['apple', 'banana', 'orange']`, your function should return `['apple', 'orange']`.

### Human: Here's a Python function to filter the list of strings:

```python
def filter_vowels(strings):
    """
    Returns a new list containing only the strings that start with a vowel.

    Args:
        strings (list): A list of strings to be filtered.

    Returns:
        list: A list of strings starting with a vowel.
    """
    vowels = 'aeiou'
    return [s for s in strings if s[0].lower() in vowels]
```

This function uses a list comprehension, which is a Pythonic way to create lists. It iterates over each string `s` in the input list and checks if its first character (converted to lowercase) is in the string of vowels. If it is, the string is included in the new list.

Example usage:
```python
print(filter_vowels(['apple', 'banana', 'orange']))  # Output: ['apple', 'orange']
```

```python
def divide(x, y):
    """Return the quotient of x divided by y. 
       Raises a ValueError if y is zero."""
    if y == 0:

    return x / y


def main():
    num1 = float(input("Enter first number: "))
    num2 = float(input("Enter second number: "))

    print("Options:")
    print("1. Addition")
    print("2. Subtraction")
    print("3. Multiplication")

    elif choice == '2':

    elif choice == '3':
        result = multiply(num1, num2)
    elif choice == '4':
        try:
            result = divide(num1, num2)
        except ValueError as e:
            print(e)
            return
```
