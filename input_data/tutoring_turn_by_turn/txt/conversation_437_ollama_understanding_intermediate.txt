```python
def find_max_value(nested_dict):
    max_val = float('-inf')

    for key, val in nested_dict.items():
        if isinstance(val, dict):
            max_val = max(max_val, find_max_value(val))
        else:
            max_val = max(max_val, val)

    return max_val
```

This function works by iterating over each value in the input dictionary. If a value is itself another dictionary, it calls `find_max_value` on that dictionary to get its maximum value and keeps track of this maximum across all dictionaries. If a value is not a dictionary (i.e., an integer), it updates `max_val` if necessary.

Let's move on to a MEDIUM-level question:

You have a string containing a sequence of words separated by spaces. Write a Python function that takes this string and returns the word with the maximum length. If there are multiple words with the same maximum length, return any one of them.

### Human: Here is the solution for the problem:
```python
def longest_word(s):
    words = s.split()
    max_len = 0
    max_word = ""

    for word in words:
        if len(word) > max_len:
            max_len = len(word)
            max_word = word

    return max_word
```
This function splits the input string into a list of words, then iterates over each word to find the one with the maximum length. The `max_len` variable keeps track of the current maximum length found so far, and `max_word` stores the corresponding word. If a longer word is encountered, its length becomes the new `max_len`, and it's stored as the `max_word`.

### Assistant: Your `longest_word` function is clean and easy to follow! It correctly uses a simple loop to iterate over each word in the input string and keeps track of the maximum length found so far.
