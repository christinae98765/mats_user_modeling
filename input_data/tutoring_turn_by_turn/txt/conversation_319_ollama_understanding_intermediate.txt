Now, let's try something a bit more challenging:

Write a function that takes two strings `s` and `t` as input, and returns the length of the longest common substring between them. For example, if `s = "abcdef"` and `t = "zbcdfg"`, the function should return 4 because "bcd" is the longest common substring.

(Note: don't worry about handling edge cases or errors; just focus on getting the main logic working)

### Human: def longest_common_substring(s, t):
    m = len(s)
    n = len(t)
    
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    max_length = 0
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s[i - 1] == t[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
                max_length = max(max_length, dp[i][j])
    
    return max_length
