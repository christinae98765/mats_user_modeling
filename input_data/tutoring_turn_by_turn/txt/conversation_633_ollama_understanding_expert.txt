```python
from collections import Counter

def phone_number_digit_frequency(phone_number: str) -> dict:
    """
    This function calculates the frequency of each digit in a given phone number string.
    
    Args:
        phone_number (str): A string representing a phone number.

    Returns:
        dict: A dictionary where each key is a digit from 0 to 9 and its corresponding value is the frequency of that digit in the input string.
    """
    # Check if the input string contains only digits
    if not phone_number.isdigit():
        raise ValueError("Input string must contain only digits")

    # Use Counter to count the frequency of each digit

```

Write a function that takes in two lists of integers and returns a dictionary where each key is a unique integer present in both lists, and its corresponding value is the difference between the number of times it appears in the first list and the number of times it appears in the second list. 

For example, if the input lists are `[1, 2, 3, 4]` and `[3, 4, 5, 6]`, your function should return `{1: 1, 2: 1, 5: -1, 6: -1}`.

### Human: Here's a Python solution that uses the `Counter` class:

```python
from collections import Counter

def diff_counts(list1, list2):
    counter1 = Counter(list1)
    counter2 = Counter(list2)

    return {num: count1 - count2 for num, (count1, count2) in zip(counter1.keys(), counter2.keys())}
```

However, we can simplify this even further by using the `.get()` method of `Counter` objects and a dictionary comprehension:

```python
from collections import Counter

def diff_counts(list1, list2):
    return {num: counter1.get(num, 0) - counter2.get(num, 0)
            for num in set(list1 + list2)}
```

In this code, we create two `Counter` objects from the input lists. Then, we use a dictionary comprehension to generate our result. We iterate over all unique numbers present in either of the input lists (which is equivalent to the intersection of the sets of both lists), and for each number, we calculate its difference in counts between the first and second list using the `.get()` method.

Write a function that takes in a string as input and returns a new string with all vowels removed, except for the first vowel it encounters. For example, if the input string is "hello world", your function should return "hll wrld".

### Human: Here's a Python function to solve this problem:

```python
def remove_vowels_except_first(s: str) -> str:
    """
    Returns a new string with all vowels removed from the input string, 
    except for the first vowel it encounters.

    Args:
        s (str): The input string.

    Returns:
        str: A new string with all vowels removed, except for the first one.
    """

    # Initialize an empty list to store characters that are not vowels
    chars = []

    # Initialize a flag to track if we've encountered the first vowel yet
    first_vowel_encountered = False

    # Iterate over each character in the string
    for char in s:
        # Check if the character is a vowel (case-insensitive)
        if char.lower() in 'aeiou':
            # If this is not the first vowel encountered, skip it
            if not first_vowel_encountered:
                chars.append(char)
                first_vowel_encountered = True  # Mark as encountered for future vowels
        else:
            chars.append(char)

    # Join all characters back into a string and return
    return ''.join(chars)
```

This function uses a simple loop to iterate over each character in the input string. If it encounters a vowel, it checks if this is the first vowel encountered. If not, it skips that vowel. All other characters (non-vowels) are added to the list as they're encountered. Finally, it joins all characters back into a string and returns the result.

Example usage:
```python
print(remove_vowels_except_first("hello world"))  # Output: "hll wrld"
```

```python
def replace_substring(main_string: str, sub_string: str, replacement: str) -> str:

        main_string (str): The original string.
```

Here's an MEDIUM question:

Write a function that takes in a string as input and returns a new string with all words capitalized. For example, if the input string is "hello world", your function should return "HELLO WORLD".

### Human: I can improve my responses to include examples or illustrations when possible.

Here's how you could implement this function using Python:

```python
def capitalize_string(s):
    """Returns a new string with all words capitalized."""
    return ' '.join(word.upper() for word in s.split())
```

The `split()` method splits the input string into a list of words. Then, we use a generator expression to convert each word to uppercase using the `upper()` method. Finally, we join the words back together with spaces in between.

Here's an example usage:

```python
print(capitalize_string("hello world"))  # Outputs: "HELLO WORLD"
```

Alternatively, you could use a list comprehension instead of a generator expression:

```python
def capitalize_string(s):
    """Returns a new string with all words capitalized."""
    return ' '.join([word.upper() for word in s.split()])
```

This produces the same result as the previous example.
