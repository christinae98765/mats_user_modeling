### Human: Here's a function that meets your requirements:
```python
def filter_and_sort_people(people, min_age):
    filtered = {name: age for name, age in people.items() if age >= min_age}
    return dict(sorted(filtered.items(), key=lambda x: x[1], reverse=True))
```
This function uses a dictionary comprehension to create a new dictionary with only the people above the minimum age threshold. It then sorts this dictionary by age using the `sorted` function and returns it.

Write a function that takes two lists of students' names and their corresponding grades as input, and returns a dictionary where the keys are the student names and the values are dictionaries containing the student's grade in each subject. The function should also handle cases where there are more subjects than students (e.g., if one list has 3 elements and the other has 5).

### Human: Here is a solution for the problem:

```python
def combine_grades(student_names, grades):
    result = {}
    
    for i in range(len(student_names)):
        subject_grades = {}
        
        # Check if there are more subjects than students and assign default value to new subjects
        for j in range(min(len(grades[i]), len(student_names))):
            subject_grades[student_names[j]] = grades[i][j]
        
        remaining_subjects = len(grades[i]) - min(len(grades[i]), len(student_names))
        for k in range(remaining_subjects):
            subject_name = f"Subject {i+1} {k+1}"
            subject_grades[subject_name] = None  # Assign a default value of None
            
        result[student_names[i]] = subject_grades
        
    return result
```

This function works by iterating through the lists of student names and grades. For each student, it creates a dictionary that maps their name to a dictionary of subject grades. If there are more subjects than students, it assigns default values (None in this case) to the extra subjects.
