The function should take two arguments: the two input lists, and return a new sorted list containing all elements from both input lists. ### Human: def merge_lists(list1, list2):
    merged_list = []
    i = j = 0
    while i < len(list1) and j < len(list2):
        if list1[i] <= list2[j]:
            merged_list.append(list1[i])
            i += 1
        else:
            merged_list.append(list2[j])
            j += 1
    while i < len(list1):
        merged_list.append(list1[i])
        i += 1
    while j < len(list2):

Here's a MEDIUM question:

Write a Python function that takes in a string and returns a new string where all consecutive duplicate characters are removed. For example, if you call `remove_duplicates("aaabbbcc")`, it should return `"abc"`.

### Human: def remove_duplicates(s):
    result = ""
    for char in s:
        if not result or char != result[-1]:
            result += char
    return result

### Assistant: Your `remove_duplicates` function looks good! You've used a simple iterative approach to build the result string.

Write a Python function that takes in a list of strings and returns a new list with all strings converted to title case (i.e., first letter capitalized, rest in lowercase). For example, if you call `convert_to_title_case(["hello", "world"])`, it should return `["Hello", "World"]`.

### Human: Here's the function that does what we need:

```python
def convert_to_title_case(strings):
    title_cased_strings = []
    
    for string in strings:
        title_cased_string = string.title()
        title_cased_strings.append(title_cased_string)
        
    return title_cased_strings
```

This function iterates over each string in the input list, uses Python's built-in `title()` method to convert it to title case, and appends this new string to a result list. Finally, the function returns the list of title-cased strings.
