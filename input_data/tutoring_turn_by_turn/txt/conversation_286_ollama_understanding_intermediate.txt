    merged = {}
    for key in set(dict1) | set(dict2):

Write a function called `flatten_nested_dict` that takes a nested dictionary as input and returns a new dictionary where all keys are at the top level, and values associated with common prefixes are grouped together. For example, if you have `{a: {b: 1}, c: {d: 2}}`, it should return `{a_b: 1, c_d: 2}`.

(Note that this function will recursively traverse the nested dictionary structure.)

### Human: def flatten_nested_dict(nested_dict):
    result = {}
    for key, value in nested_dict.items():
        if isinstance(value, dict):
            sub_dict = flatten_nested_dict(value)
            for sub_key, sub_value in sub_dict.items():
                result[key + '_' + sub_key] = sub_value
        else:
            result[key] = value
    return result
