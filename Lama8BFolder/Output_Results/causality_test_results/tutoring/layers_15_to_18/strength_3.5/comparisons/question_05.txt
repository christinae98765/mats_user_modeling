**Exercise: Rock, Paper, Scissors Game**

Create a simple Rock, Paper, Scissors game in Python. The game should:

1. Ask the user for their choice (rock, paper, or scissors)
2. Generate a random choice for the computer
3. Determine the winner based on the game's rules:
   - Rock beats scissors
   - Scissors beats paper
   - Paper beats rock
4. Display the result of the game

**Example Use Case:**

```
User: rock
Computer: scissors
Result: User wins!
```

**Your Task:**

Write the Python code to implement this game.

**Hint:** You can use the `random` module to generate a random choice for the computer.

**Get started!**

```python
import random

# Define the choices
choices = ['rock', 'paper', 'scissors']

# Get the user's choice
user_choice = input("Enter your choice (rock, paper, or scissors): ")

# Generate a random choice for the computer
computer_choice = random.choice(choices)

# Determine the winner
if user_choice == computer_choice:
    result = "It's a tie!"
elif (user_choice == 'rock' and computer_choice == 'scissors') or \
     (user_choice == 'scissors' and computer_choice == 'paper') or \
     (user_choice == 'paper' and computer_choice == 'rock'):
    result = "User wins!"
else:
    result = "Computer wins!"

# Display the result
print(f"User: {user_choice}")
print(f"Computer: {computer_choice}")
print(result)
```

**Requirements:**
---------------

*   The program should use a random number generator to select a number between 1 and 10.
*   The user should be able to guess the number within 5 attempts.
*   The program should display the number of attempts remaining after each guess.
*   The program should display a success message if the user guesses the number correctly.
*   The program should display a failure message if the user guesses the number incorrectly and runs out of attempts.
*   The program should display a message indicating the correct number if the user guesses it correctly.
*   The program should display a message indicating the number of attempts taken if the user guesses it correctly.
*   The program should use a loop to allow the user to play the game multiple times.
*   The program should use a random number generator to select a new number for each game.
*   The program should use a try-except block to handle invalid input.
*   The program should use a clear and concise code structure.
*   The program should be well-documented and easy to understand.

**Code:**
------

```python
import random

def play_game():
    """Play a guessing game."""
    for _ in range(5):
        number = random.randint(1, 10)
        attempts = 5
        print(f"Your guess: {number} (in {attempts} attempts)")
        while attempts > 1:
            user_guess = input("Enter your guess: ")
            try:
                user_guess = int(user_guess)
                if user_guess == number:
                    print(f"Congratulations! Your guess was correct. It took you {attempts} attempts.")
                    return
                else:
                    attempts -= 1
                    print(f"Your guess was incorrect. Remaining attempts: {attempts}")
                if user_guess < number:
                    print("Your guess was too low.")
                else:
                    print("Your guess was too high.")
            except ValueError:
                print("Invalid input. Please enter a valid number.")
        print(f"Your guess was incorrect. The correct number was {number}.")
    print("You ran out of attempts. The correct number was {number}.")
```
